## WHAT:三维评判顺序

| 维度 | 英文 | 评分粒度 | 地位 |
|---|---|---|---|
| **流利度** | Fluency | 4 档 | 第一判断标准 —— 最先看,直接决定信息量 |
| **规范性** | Normativity | 4 档 | 第二判断标准 —— 决定信息质量,中国学员关键区分维度 |
| **词汇使用** | Vocabulary | 2 档 | 锦上添花 —— 只在级别摇摆时参考 |

**关键认知**:这三个维度**不是并列的**,而是**有先后顺序**——先看流利度,再看规范性,词汇只在定不下级别时才考虑。

## 维度一:流利度(Fluency)

> 你对英语的**熟悉程度**。最先判断,**反映在英语上花了多少功夫**。

### 4 档评分

| 档位 | 表现 |
|---|---|
| **1** | 语速很快,很流利 |
| **2** | 语速适中,有节奏的停顿 |
| **3** | 语速偏慢,但可以持续输出 |
| **4** | 语速很慢,停顿很多,信息量很少 |

### 简化版(粗判 A/B/C)

学员一张嘴,基本就能判断在欧标的哪个大类别:

- **A 级别**:讲话很磕绊,停顿很多
- **B 级别**:语速适中
- **C 级别**:讲得贼溜

### 为什么流利度排第一

流利度反映**神经通路建立程度**。磕绊 = 大脑还在翻译,没有建立直接通路;流利 = [[MCES模型|M→S]] 的口语路径已经走通。

口语测试里**固定时间内能传递多少信息**,首先取决于流利度。

## 维度二:规范性(Normativity)

> 你能否正确说出**结构完整**的句子,尤其是**复杂句**。

### 4 档评分

| 档位 | 表现 |
|---|---|
| **1** | 几乎没有词性错误,且能使用复杂句式 |
| **2** | 几乎没有词性错误,但只能使用简单句 |
| **3** | 存在部分句式和词性错误 |
| **4** | 存在大量词性错误,无法造句 |

### 为什么叫"规范性"不叫"语法"

标准口语考试这个维度叫**Grammar**,但 Barry 更倾向用"**规范性**"——更形象,也更直接对应**中国学员的两类典型错误**:

1. **固定搭配错用**(尤其是固定句型)
   - 例:"We need to **discuss about** this"(应为 "discuss this")
   - 例:"I'm **agree with** you"(应为 "I agree with you")
   - 很多学员**无法正确讲出一个复杂句式**

2. **词性错用**(名词 / 形容词 / 副词形式混用)
   - 例:"Your **success** is great" vs "You are **successful**"
   - 例:"Speak **loud**" vs "Speak **loudly**"

### 简化版(粗判 A/B/C)

实战中:

- **A 级别**:较多词性错误,基本无法造句只能蹦词,即便造句也只是简单句(如 "We should do something. / We need to do something.")
- **B 级别**:少量词性错误,简单句熟练,偶尔使用复杂句式
- **C 级别**:几乎无词性错误,熟练使用复杂句式

### Q&A:不是说"先练对再练快"吗?

> **Q**:Barry,你平时不是说要先保证**练对**,再**练快**吗?

> **A**:是的。所以**评级时反着来——先看流利度,再看规范性**。
> 因为几乎没有某个学员讲得很流利,但全是语病的情况。**99% 的情况是,你讲的不标准的时候,你是无法加速的**。
> 所以通过流利度做基本盘判定,然后通过规范性微调。这是实战中总结出来的经验。

## 维度三:词汇使用(Vocabulary)

> 你的表达中**是否有高级词汇**。锦上添花的标准,**只在级别之间摇摆时参考**。

### 2 档评分

| 档位 | 表现 |
|---|---|
| **1** | 会使用高级词汇 |
| **2** | 没有高级词汇 |

**高级词汇**包括:
- 日常词汇的高级替换(good → extraordinary;important → crucial / significant)
- 职场地道表达(XFN / one-on-one ...)

### 为什么排在最后

词汇好但流利度 / 规范性不到位 = 花架子。前两个维度分别决定**信息数量**和**信息质量**——这是基本盘。

但**这并不代表高级词汇不重要**——就像中文能不能用上成语典故,是**长线积累的储备**。

### 典型场景

定不了"这学员是 B+ 还是 C-"时,看词汇:
- 只会高频基础词 → 倾向 B
- 能用 collocation / phrasal verbs / idiomatic 表达 → 倾向 C

## 三维组合 → CEFR 映射表

按 **流利度档-规范性档-词汇档** 组合判级:

### C 级

| 组合 | 级别 |
|---|---|
| 1-1-1 | C1 |
| 2-1-1 | C1 |
| 1-1-2 | C1 or B2 |

### B 级

| 组合 | 级别 |
|---|---|
| 1-2-1 | B2 |
| 2-1-2 | B2 |
| 2-2-1 | B2 |
| 1-2-2 | B1+ |
| 1-3-1 | B1 |
| 1-3-2 | B1 |
| 2-2-2 | B1 |
| 2-3-1 | B1 |
| 2-3-2 | A2 or B1 |

### A 级

| 组合 | 级别 |
|---|---|
| 2-4-1 / 2-4-2 | A2 |
| 3-2-/ / 3-3-/ | A2 |
| 3-4-1 / 3-4-2 | A1 |
| 4-/-/ | A1 |

### 现实不存在的组合

- **1-4-1 / 1-4-2**(语速很快但全是语病)—— 不存在
- **3-1-1 / 3-1-2**(语速偏慢但能用复杂句式)—— 不存在

→ 印证了 **Q&A** 里"99% 的情况是,讲的不标准就无法加速"。

## 不进维度:连贯性 / 发音

还有两个维度**没有进入评判表**:

| 维度 | 英文 | 为什么不进 |
|---|---|---|
| **连贯性** | Coherence | 学员基本逻辑都没问题,区分度低 |
| **发音** | Pronunciation | 学员发音都能听懂,区分度低 |

> Barry 的看法:这俩比高级词汇还要锦上添花。**只有再往 C 级别走,才会用上这两个指标**。实战中几乎不用,所以也就没体现在评判标准中。

## HOW:从评判倒推训练

### 训练优先级

1. **先抓规范性**
   - 刻意总结**固定表达和句型**
   - 使用这些"**[[语块概念-Meaning-Unit|语块]]**",先搭建简单句,再尝试复杂句式(定语 / 状语 / 名词性从句)
   - 不会语法的同学去补 [[语法-跨听口支撑]]

2. **同步推进熟练度**
   - 通过 [[视译训练-Sight-Interpreting|视译]]练到"**脱口而出**"
   - 看到中文直接说英文,甚至想到**意思**就能脱口而出成串的英文——所谓"英语思维"就是这么来的

3. **做到以上两点 = B2 级别,完虐 90% 的人**

4. **往 C 级别突破**:
   - 积累**高级表达**,替换日常词汇
   - 积累**思维框架**,广泛阅读,提升思维敏捷度
   - 关注**发音 / 连读 / 语音语调**,模仿 native speaker

### 和 CEFR 的对应

三维定级映射到 CEFR:

| Barry 三维粗判 | 大致对应 CEFR |
|---|---|
| A(磕绊 + 蹦词) | A1 - A2 |
| B(语速适中 + 简单句熟练) | B1 - B2 |
| C(流利 + 复杂句准确) | C1 - C2 |

详细对应见上方"三维组合 → CEFR 映射表"和 [[CEFR分级训练方案]]。

## 评判实操:Mini-Presentation 定级

定级的主场景是 [[Mini-Presentation训练方法论]]——**3 分钟足够看出一个学员整体水平**。

不需要分阶段听某个维度,三维同时在跑,听完 3 分钟脑子里就能落到一个组合(流利度档-规范性档-词汇档),对照映射表得出 CEFR 级别。
