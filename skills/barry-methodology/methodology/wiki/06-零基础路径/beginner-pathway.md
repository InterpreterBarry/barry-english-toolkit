# 零基础英语学习路径

> 本文档面向**纯粹零基础**的成人学习者（从未接触过英语，或只会 ABC 和 Hello 的程度）。
> Barry 主体系（MCES 模型、语块优先、视译输出）面向的是已有基础的中国职场人，不适用于零基础。本路径是零基础到"有基础"之间的桥梁。

**核心参考来源**：Gabriel Wyner《Fluent Forever》Stage 1-2（读书笔记见 [[Fluent Forever - Book Notes]]），结合 Barry 方法论调性和学术证据进行调整。

---

## Stage 0：认知启动（1-2 周）

**目标**：建立"英语是一套声音系统"的基本认知，不急着记单词。

**内容**：
- 学 26 个字母不能只学名字（A = "诶"、B = "必"），更重要的是学它们在单词中的发音规律——比如 A 在 cat 里是 /æ/，在 cake 里是 /eɪ/，在 car 里是 /ɑː/。这就是**自然拼读（Phonics）**的基本逻辑。
- 感受英语的基本节奏：重音在哪里、哪些音弱读、句子的起伏感
- 不要求记住任何单词，只要求"耳朵开始习惯英语的声音"

**工具**：字母歌 + 简单的节奏模仿（跟读 chant）

**注意**：成人零基础学 Phonics 和儿童不同——儿童是已经会说、要学认字（sound→letter）；成人是反过来，看得懂拼写但不确定怎么发音（letter→sound）。方向不同，但核心逻辑一样：建立"看到字母组合→知道怎么读"的条件反射。

---

## Stage 1：发音地基（3-4 周）

**目标**：校准耳朵 + 嘴巴，搞定英语核心音素。

### 1.1 Minimal Pairs 听辨训练

**原理**：两个词只差一个音（ship vs sheep），反复让学习者判断"听到的是哪个"。大脑被迫注意微小差异 → 注意到 → 能分辨 → 能模仿 → 能稳定产出。

**学术支撑**：
- Logan et al. (1991) 奠基实验：日语母语者通过 68 组 /r/ vs /l/ minimal pairs 训练，显著提升辨音准确率
- Cambridge HVPT 元分析：效果量中到大，且训练效果有长期保持
- 用多个说话人的录音效果更好（High Variability Phonetic Training），让大脑学会从不同口音中提取同一个音素特征

**训练三步走**：

| 步骤 | 做什么 | 要点 |
|---|---|---|
| Step 1：感知 | 播放录音，从两个选项里选"听到的是哪个" | 必须有即时反馈（对/错），不能光听不判断 |
| Step 2：产出 | 自己说两个词，感受嘴巴位置差异 | 可以用镜子观察嘴型变化 |
| Step 3：语境 | 把音放进简单句子里练 | "Is this a ship or a sheep?" |

**中国学习者重点 Minimal Pairs**：

| 难点 | 举例 | 中国学习者的问题 |
|---|---|---|
| /ɪ/ vs /iː/ | ship / sheep, bit / beat | 中文没有长短元音区分 |
| /æ/ vs /e/ | bad / bed, man / men | 中文没有 /æ/ 这个音 |
| /l/ vs /r/ | light / right, led / red | 部分方言区分不清 |
| /θ/ vs /s/ | think / sink, thick / sick | 中文没有咬舌音 /θ/ |
| /v/ vs /w/ | vine / wine, vet / wet | 中文没有咬唇音 /v/ |
| /n/ vs /ŋ/ 尾音 | sin / sing, thin / thing | 中文有但位置规则不同 |
| /ɒ/ vs /ʌ/ | hot / hut, cop / cup | 中文没有这个对立 |

**工具推荐**：ELSA Speak、Sounds of Speech（App 类，自带录音+即时反馈）

### 1.2 IPA 国际音标基础

- 不求全部记住，但要知道每个音素的**发音位置逻辑**（舌头在哪、嘴唇怎么摆、声带振不振动）
- IPA 是 Barry 体系的标配（[[Fancy Vocab]] 每个词都有 IPA 列），零基础阶段就开始建立这个习惯

### 1.3 自然拼读（Phonics）进阶

- 系统学习常见字母组合的发音规则
- 例：ough 在不同词里的发音（though /əʊ/, through /uː/, tough /ʌf/, thought /ɔː/）
- 目标：看到一个陌生单词，能大致猜出怎么读

## Stage 2：基础词汇网络（6-8 周）

**目标**：建立 300-500 个高频基础词的认知网络。

> 词汇量参考依据：Nation (2006) 数据显示 1000 词族→覆盖日常 85%。300-500 是零基础起步的第一个里程碑，不是终点。

### 学什么词？

优先级：
1. **具象名词**：身体部位、食物、动物、日常物品、颜色、数字
2. **基础动词**：go, come, eat, drink, see, hear, want, like, have, make
3. **核心形容词**：big, small, hot, cold, good, bad, new, old, happy, sad

### 怎么学？

**图片 + 发音 + 中文释义 三件套**：
- 不走极端的"完全不翻译"（Wyner 的方式），但鼓励**先看图再看翻译**
- 每个词配发音 + IPA 标注
- 图片法对具象词有效，但遇到抽象词（big, happy, because）不能只靠图片，必须加中文释义确保理解准确

**个人连接（Personal Connection）**：
- 每个词都要找到和自己生活的连接点
- "你第一次见到这个东西是什么时候？""这个东西在你家哪里？"
- 个人连接越强，记忆越牢（加工深度原理）

**SRS 间隔重复**：
- 每天 20-30 分钟闪卡复习
- 在快要忘但还没完全忘的时候复习，记忆效率最大化

## Stage 3：从单词到语块过渡（4-6 周）

**目标**：从"一个一个词"过渡到"一组一组表达"。

### 基础句型框架

先掌握最高频的句子骨架：
- I want to...（我想要...）
- Can I...?（我可以...吗？）
- This is...（这是...）
- I like / I don't like...（我喜欢/不喜欢...）
- Where is the...?（...在哪里？）
- How much is...?（...多少钱？）

### 高频生存语块

不用拆解语法，整块记住直接用：
- How much is this?
- Where is the bathroom?
- I don't understand.
- Can you say that again?
- Nice to meet you.
- Thank you very much.

### 从单词卡到短句卡

- 闪卡的最小单位从单词升级为短句/语块
- 正面：中文意思或场景描述
- 背面：英文语块 + IPA + 录音

### 毕业标志

当学习者能用 3-5 个词造出自己的句子时（哪怕语法不完美），说明已经具备了进入下一阶段的基础。

## Stage 4：衔接 Barry 主体系

**前提条件**：
- 基础词汇 300-500 个
- 能听懂简单日常对话的大意
- 能用简单句表达基本需求

**衔接方式**：
进入 Barry 主体系 → [[MCES模型|MCES 模型]] → 语块优先 → 视译训练 → 输出驱动（完整方法论见 [[Barry职场英文沟通方法论_完整版|Barry职场英文沟通方法论]]）

**本质转换**：
从"从零建楼"模式 → 切换到"装修+加建"模式。

详见 → [[methodology|核心方法论速查]]
