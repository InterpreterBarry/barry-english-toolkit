# 2.1 学习心态:没有捷径

> 打掉迷思 1/2/3 之后,**正确的学习心态**:每天坚持 + 周期性 + 习惯化。**用官方学时参考管理预期**(俗称"欧标小时数"),**用隧道理论保持坚持**——本质就是"**学英文像过隧道,坚持下去就能看到曙光**"。

## 三大原则 ⭐

| 原则 | 说明 |
|---|---|
| **每天坚持** | 最少 30 分钟练习 |
| **周期性** | 3-6 个月为一个周期,**每 3 个月可见进步** |
| **习惯化** | 必须融入日常生活,**成为习惯**——不是"任务",是"吃饭" |

**配套落地**:每日训练必须是 [[输入输出并行原则|学练并行]]——刻意学语言形式(LFL)和实战听说(MFI + MFO)两条线同时跑,不是只背不用,也不是只泛听泛说。详见 [[Four-Strands-总览]]。

## 官方学时参考:管理你的预期 ⭐

> 俗称"欧标小时数",但**欧标(CoE)官方从未发布过学时数**。最权威的逐级数据来自 **Cambridge English《How long does it take to learn a foreign language?》(2018)** 第 10 页 **Example 1:良好学习环境中的成年学习者**(高动机 + 好资源 + 母语与英语同文字系统)。

**Cambridge 官方逐级学时**(guided learning hours,从上一级升到该级别)+ **累计**(从零到该级别共需):

| 晋级阶段 | 逐级增量 | 累计 |
|---|---|---|
| 零基础 → A1 | 90–100h | 90–100h |
| A1 → A2 | 100–150h | 190–250h |
| A2 → B1 | 160–240h | 350–490h |
| **B1 → B2** | **180–260h** | **530–750h(到 B2 能正常工作沟通)** |
| B2 → C1 | 200–300h | 730–1,050h |
| C1 → C2 | 300–400h | 1,030–1,450h |

三个口径细节:
- 这是 Cambridge 给"**良好学习环境的成年学习者**"的情境模型,**不是适用所有人的统一标准**——而且级别越高、单级越费时(C1→C2 要 300–400h)
- guided learning hours = **有指导的课堂学时**(含布置的作业/任务),不等于全部接触量
- ⚠️ **中文母语者还要上调**:这个模型假设"母语与英语**同文字系统**";中文用汉字、不满足这条,从零学官方口径下只会更高(ef.com 的对应说法是母语差异大者**再加 50%**)

### 官方表 vs 繁星实测 ⭐

**官方表假设的是"零基础课堂学习者从头积累";繁星学员不是这个画像**:

- 你有 6+ 年学校英语底子,**被动储备早已在仓**——官方表里的大几百小时,学生时代已经付过了
- 训练做的是**激活**(被动 → 主动,见 [[被动词汇vs主动词汇]]),不是从零搭建
- 所以**实测升级速度显著快于官方表**——"几个月可见一个台阶"是带方法、对症训练下的真实观察

两轨各管一件事:**官方表管总量诚实**(语言能力的物理总量在那里,没有捷径——而且 Cambridge 这个数还是"同文字系统"的乐观值,中文母语者从零只会更高);**繁星实测管预期**(你的起点远不是零,所以你能快是因为**存量**,不是因为路短)。

### 怎么用这张表

**做心理预期管理**:

- 你每天 30 分钟 = 一周 3.5 小时 = **一年 ~180 小时**
- 按 Cambridge 增量,A2 → B1(160–240h)≈ **1–1.5 年**;A2 → B2(340–500h)≈ **2–3 年**
- 带方法 + 存量激活,繁星实测普遍快于这个数

**关键认知**:**这不是"打击你",是"让你看清现实"**——
- ❌ "我 3 个月学不会就放弃了" → 3 个月的小时数撑不起一个级别跨越
- ✅ "我每天 30-60 分钟,坚持 1-2 年,从 A2 上 B2" → 现实可行的目标

> **3 周看不到进步就放弃 ≠ 你没天赋,是周期还没到**。

## 隧道理论:坚持的本质 ⭐

> **学英文就像过隧道,很长的一段时间你都身处黑暗中,但如果你愿意坚持前行,总有一天可以看到曙光。**

**核心比喻**:

```
入口 ─────[漫长的黑暗]──────── 出口(曙光)
   ↑                                ↑
   学的越多,                      实战中能用的
   感觉不会的越多                  内容越来越多
   (圆圈边界拓展)                  (高频内容反复出现)
```

### 两个关键认知

1. **掌握的越多,感觉不会的就越多** —— 圆圈不断拓展,边界感越来越强。这不是退步,是认知扩展
2. **高频词汇和表达会反复出现** —— 坚持打卡,自然而然就会记住

> 隧道理论本质就是"**坚持**"——只是给"坚持"一个更具体的画面。

## 学员体感时间轴(繁星实测口径)

**前提:起点 A2-B1(有学校英语底子的职场人)+ 每天 30-60 分钟对症训练**。按官方增量(每级约 200h)折算,这个节奏从 A2 到 B2 本来就是 1.5-3 年量级;繁星实测普遍更快:

| 阶段 | 体感 |
|---|---|
| **第 1-3 个月** | 感觉自己进步小,**学得越多越觉得不会的多**——这是隧道入口的黑暗,正常 |
| **第 3-6 个月** | 部分高频内容开始"脱口而出"——隧道里看到第一缕光 |
| **6-12 个月** | 常见职场场景能跑下来——开始**能用上**(B1 水平) |
| **1-3 年** | 能流利沟通(B2 水平)——**真正的曙光**在这里 |

**3 个月是分水岭**:大部分人在 3 个月内放弃 —— 因为**没看到进步**。但**周期还没到**。
