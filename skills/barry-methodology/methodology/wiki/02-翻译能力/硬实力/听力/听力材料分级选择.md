# 听力材料分级选择

> 不同级别选不同材料,**练一小段、走两道工序**——贪多嚼不烂,刷量不如啃透。

## 核心定位

- 这是听力训练的**选材指南**
- 与 [[CEFR分级训练方案]] 对照(口语端),本页是听力端
- 选对材料 = 训练事半功倍;选错材料 = 磨耳朵

## WHAT:按级别选材

| 级别 | 推荐材料 |
|---|---|
| **A 级别** | 慢速听力材料(如 VOA Special English,2014 年已更名 **VOA Learning English**) |
| **B 级别** | 常速英语(BBC Learning English、TED Talks、YouTube) |
| **C 级别** | 以英文为媒介探索世界(YouTube 各类材料) |
| **职场人**(通用) | 优先选择**商业职场**相关内容 |

## WHY:为什么要分级

### 选太难的材料

- 盲听时抓不到大意
- 70-80% 听懂率达不到([[声音到文字-听写四步|听写训练]]前提)
- 结果:逐字猜谜 → 极度挫败 → 放弃

### 选太简单的材料

- 没有生词、没有陌生表达
- 大脑不需要"努力"
- 结果:磨耳朵,有时间但没增量

### 选错内容类型

- A 级学员直接上日常对话(快速、连读多、俚语)→ 干不动
- C 级学员还在刷 VOA Special English → 进步停滞

## HOW:练习频率与流程

### 基础节奏

- **每天半小时**,练一小段(**3-5 句话**)
- 不贪多,同一段榨干

### 同一材料两道工序

```
Step 1  听写训练(练"听清",声音→文字)
          ↓
Step 2  英到中视译(练"听懂",文字→意思)
```

**为什么要两道工序**:
- 只做听写:听清了但不见得懂
- 只做视译:抓意思了但声音辨识度没提升
- **两道工序**:同一材料价值被榨到 100%,远高于"每天听新内容"的泛泛输入

### 选材细节

**职场人(通用前置建议)**:
- 优先商业职场内容:corporate 访谈、MBA 讲座、商业播客、行业 YouTube
- 用场景匹配度覆盖自己的真实需求——日常工作要听的是会议 + 汇报,不是旅行英语

**A 级**:
- VOA Special English(现名 **VOA Learning English**):语速慢、词汇受限、句式简单
- 目标:建立基础声音-文字匹配库,扩大被动词汇储备

**B 级**:
- TED Talk:话题丰富、语速正常、逻辑清晰(适合练 [[Global-vs-Local-Listening|Global Listening]] 的深层)
- BBC Learning English:难度中等、材料精讲
- YouTube 商业频道:接近真实职场语速

**C 级**:
- 以英文为**媒介探索世界**,而非把英文当**学习对象**
- 行业顶级访谈、大学公开课、专家圆桌
- 接受不完全听懂——C 级听的是内容,不是听力训练

## 常见误区

- ❌ "我就是想挑战自己,直接上 CNN" — A/B 级直接上 CNN 等于听不懂硬扛,时间浪费
- ❌ "我已经 C 级了还在刷 BBC Learning English" — 材料喂养不够,进步停滞
- ❌ "每天换一段新的,积少成多" — 不榨干一段的价值,永远浮在表面

## 配合其他训练

- 选材难度由 [[听力问题诊断体系-Hearing-vs-Listening]] 决定训练类型
- 听完的材料 → 转口语训练 → [[复述-Retelling]] (B 级后)
- [[Meaning-focused-Input|泛听]] 另算,用难度更低的材料拉**暴露量**
