# 硬实力 · 总览(= 能调用储备)

> [[0-翻译能力总览|翻译能力]] 下的**主体子板块**(与 [[0-翻译技巧总览|软实力]] 平级)。**副标题 = 能调用储备**——关注焦点是"语言储备调用得出来吗",不是"听力口语能力"(后者会误导,因为软实力也涉及听口)。**抽象上 = 语言储备**(单词 / 固定搭配 / 句型);**方法论层面 = 听力口语能力如何提升**。**两层分析维度:宏观(Four Strands)+ 微观(MCES)**,微观下面再分口语和听力。

## 硬实力的两层定义

**抽象层**:**语言储备**——单词 / 固定搭配 / 句型 / 俚语等"米和弹药"。

**方法论层**:**听力口语能力如何提升**——讲具体怎么练、用什么训练方法。

## 两个分析维度 ⭐

硬实力同时从**两个并列维度**分析,各自有不同的目的:

| 维度 | 框架 | 目的 |
|---|---|---|
| **宏观维度** | [[Four-Strands-总览\|Four Strands]] | 给学员设计**整体英文提升 Solution** |
| **微观维度** | [[MCES模型\|MCES]] | (1) 理解听口背后的**原理**<br>(2) **pinpoint** 关键训练方法 |

**它们不是上下级,是同一件事的两个仪表盘。**

### 宏观维度:Four Strands(Solution 设计)

Paul Nation 的四条线框架:
- **Meaning-focused Input**(MFI)
- **Meaning-focused Output**(MFO)
- **Language-focused Learning**(LFL)
- **Fluency Development**(FD)

**落地工具**:[[Four-Strands-训练方法工具箱|CEFR(A/B/C) × Four Strands 训练方法工具箱]]——12 个训练方法 × 3 个级别,直接对应学员的 Solution 配比。

**Barry 的占比规则**(中国学员定制,不照搬 25/25/25/25):
- 水平越低 / 越 basic → 越侧重 **LFL**(词汇 / 表达 / 句型)
- 越高阶 → 越偏 **meaning-focused**(实战层面)

### 微观维度:MCES(原理 + 关键方法)

Barry 自创的认知机制框架:
- **M**(Message,意思)
- **C**(Chinese,中文)
- **E**(English,英文)
- **S**(Sound,声音)

**作用**:从微观大脑路径上,**分析底层逻辑 + 给出解决方案 + 具体训练方法**。

#### 微观维度下的两条路径

**注意**:口语和听力的切分逻辑**不同** —— 口语按**训练内容**切(三大并列模块),听力按**链路步骤**切(两步走 + 高阶训练)。这是因为两者的核心问题不同:口语关心"内容缺什么",听力关心"链路哪一步卡住"。

```
微观维度(MCES)
│
├── 口语逻辑(C→E→S 路径)── 按"训练内容"切分
│   │
│   ├── 走 MCES 链路: C → E → S
│   ├── 核心卡点: C → E
│   │
│   └── 三大并列模块
│       ├── 模块 1: 语料储备(主体)⭐
│       │   ├── 训练目标两维度:单词(抓本质含义)/ 固定搭配+句型(抓背后意思)(被动→主动)
│       │   ├── 核心训练方法:[[中英语块加视译|中英语块 + 视译]] ⭐⭐
│       │   └── 高阶训练:[[Mini-Presentation训练方法论|Mini Presentation]](B2+)
│       │
│       ├── 模块 2: 语法(跨听口支撑)
│       │   └── 详见 [[语法-跨听口支撑]]
│       │
│       └── 模块 3: 发音(S 节点)
│           └── 详见 [[发音-S节点专题]]
│
└── 听力逻辑(S→E→C→M 路径,简称 SECM)── 按"链路步骤"切分
    │
    ├── 走 MCES 反向: S → E → C → M
    │
    ├── 两个核心步骤
    │   ├── 步骤 1: 声音 → 文字
    │   │   └── 关键训练方法:[[声音到文字-听写四步|听写训练]](四步)⭐
    │   │
    │   └── 步骤 2: 文字 → 意思
    │       ├── 关键训练方法:[[文字到意思-英到中视译|英到中视译]] ⭐
    │       │   └── 在 [[顺句驱动]] 原则下按意群一段段翻译
    │       └── ⚠️ 强依赖 [[语法-跨听口支撑|语法支撑]](意群切分靠句子成分)

```

## 语法的特殊定位 ⭐

**语法不是独立板块,而是分散在三处出现的支撑能力**(详见 [[语法-跨听口支撑]]):

```
1. 微观层面
   ├── (a) 口语:语料储备 / 语法 / 发音 三大并列模块之一
   ├── (a) 听力:「文字到意思」链路的核心支撑
   └── (b) 长难句分析 三步走

2. Four Strands 框架
   └── Language-focused Learning(LFL) 的重要内容
```

**为什么这么处理**:语法本身是**支撑能力**,不是终极目标。所有语法投入最终都为"听口能力"和"长难句理解"服务——所以放在使用场景里讲,而不是独立成板块。

## 硬实力训练 vs 软实力训练

**同一个训练动作(如视译),关注焦点不同 → 训练的是不同实力**:

| 关注焦点 | 是哪类训练 |
|---|---|
| 单词 / 表达 / 句型用得**准不准、地不地道** | 硬实力训练 |
| **能不能自主处理信息** + 切分脱壳 + 逻辑显化 + 讲人话 + **沟通结果完成** | 软实力训练 |

**关键洞察**:训练动作不需要新的——需要的是**关注焦点的切换**。
