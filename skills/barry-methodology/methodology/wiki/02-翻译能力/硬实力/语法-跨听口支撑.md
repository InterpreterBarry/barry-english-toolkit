# 语法 · 跨听口支撑

> **语法不是独立板块,而是分散在三处出现的支撑能力。** 有独立 wiki 页作为知识参考,但在整体框架结构里不占独立位置。**核心原则**:大部分内容当固定搭配背 / 少部分需要专项学习——避免在语法上过度投入,实战为导向。

## 整体定位:语法在哪里出现 ⭐

语法分散在 **3 个位置**出现,不单独占一个板块:

```
语法的三个分布位置(分散引用,不单独占位)
│
├── 1. 微观层面
│   │
│   ├── (a) 听力和口语
│   │   ├── 口语:语料储备 / 语法 / 发音 三大并列模块
│   │   └── 听力:[[文字到意思-英到中视译|文字到意思]] 链路的核心支撑
│   │           (切意群、分析复杂句的句子成分)
│   │
│   └── (b) 长难句分析 三步走(详见本页 ↓)
│
└── 2. Four Strands 框架
    └── [[Language-focused-Learning|LFL]] —— 语法是 LFL 的重要内容
```

**为什么不单独成一块**:语法本身是**支撑能力**,不是终极目标。所有语法投入最终都为"听口能力"和"长难句理解"服务。它的价值在被使用时才显现——所以放在使用场景里讲,而不是独立成板块。

## 核心原则:两分法 ⭐

```
语法内容
├── 大部分 ──> 当固定搭配背诵
└── 少部分 ──> 需要专项学习(背不了)
```

这是 Barry 独到的**语法学习哲学**——**避开了大多数人在语法上的过度投入**。

## 必须专项学习的内容(背不了)

### 1. 时态

**用错母语者会时空错乱**——所以时态必须懂得它的**构成原理**,不能死记。

时态不是某个固定搭配,它是**整套时间表达系统**:
- 一般现在 / 一般过去 / 一般将来
- 现在进行 / 过去进行 / 将来进行
- 现在完成 / 过去完成 / 将来完成
- 现在完成进行 / 过去完成进行 / 将来完成进行

学员必须知道**何时用哪个**,不能死背搭配。

### 2. 句子成分

**主谓宾定状补**——掌握后才知道**英文句子是怎么造出来的**。

这是英文句子的"骨架"——知道骨架,才能在拼句子时知道**每个成分往哪放**。详见 [[中英语块加视译]] 里"自己拼句子的过程"的精髓——拼句子的前提是懂骨架。

**句子成分还是长难句分析三步走的底层依赖**(详见本页"长难句分析三步走"节)。

### 3. 复杂句与从句

需要单独学习的从句类型:
- 名词性从句(主语从句 / 宾语从句 / 表语从句 / 同位语从句)
- 定语从句
- 状语从句

### 4. 被动语态 ⭐

**这条容易归错**——被动语态**也要专项学,不是当固定搭配背**。

被动语态涉及结构变换(主动 → be + 过去分词 + by...),需要**真正理解结构机制**才能灵活使用。

## 可以当固定搭配背诵的内容

> 这些内容**不要纠结每个词为什么这么排**,直接整体当**搭配/句型**记。

| 内容 | 性质 |
|---|---|
| **虚拟语气** | *If I were you... / I wish I could...* 等结构当整体记 |
| **倒装结构** | *No sooner had I... than... / Not only..., but also...* 当整体记 |
| **强调结构** | *It is X that... / It was X that...* 当整体记 |

**为什么这些可以当搭配记**:它们的核心价值是"**这种语法结构对应这种意思**",而不是"理解每个词的角色"。直接背整体反而**调用更快**。

## 长难句分析三步走 ⭐(英到中理解的语法应用)

这是"语法作为支撑能力"最典型的应用场景——**用语法分析能力,搞定英文长难句的理解**。

### 三步走流程

```
长难句分析三步走
│
├── 第一步:通读
│   └── 尽全力理解大意,过生词
│
├── 第二步:分析成分(语法层面)
│   ├── (a) 找母句(剥洋葱找主谓宾骨架)
│   ├── (b) 拆从句(几个从句,分别什么类型)
│   └── (c) 拆每一个成分(用二维矩阵对号入座)
│
└── 第三步:做视译
    ├── (a) 划分意群(成分识别提供候选边界,按意思的完整性决定在哪里落刀)
    └── (b) 顺句驱动(按原句顺序,一个意群一个意群翻译,加连接词形成地道中文)
```

### 第一步:通读

**目标**:尽全力理解大意,扫一遍生词。

**这一步不需要完美理解**——只是建立大致语义框架,知道这句话在讲什么主题。生词扫一遍即可,不必查到位。

### 第二步:分析成分(语法层面)

**目标**:把句子的骨架剥出来,从句和成分都认清楚。

#### (a) 找母句(剥洋葱)

剥掉所有从句、修饰语,找出**最核心的主谓宾骨架**——这是整句话的"主干"。

#### (b) 拆从句

列出这句话里**有几个从句**,**分别是什么类型**(名词性 / 定语 / 状语)。注意从句之间可能有**嵌套关系**——一个从句嵌在另一个从句里。

#### (c) 拆每一个成分

把每个成分对号入座到"**8 大成分位置 × 4 种砖块**"的二维矩阵里。

### 前置知识:8 大成分位置 + 插入语(独立成分)

英语再长的句子,都是这 8 个位置的排列组合:

| 位置 | 角色 | 例子 |
|---|---|---|
| 主语 | 谁/什么 | **The boss** sent an email. |
| 谓语 | 做什么 | The boss **sent** an email. |
| 宾语 | 对谁/什么 | The boss sent **an email**. |
| 定语 | 修饰名词 | The **new** boss sent an **urgent** email. |
| 状语 | 修饰动词/全句 | The boss **quickly** sent it. |
| 表语 | 系动词后 | The email is **urgent**. |
| 宾补 | 补足宾语 | They called the email **urgent**. |
| 同位语 | 解释名词 | The boss, **Tim**, sent it. |
| 插入语(独立成分,不占 8 大位置) | 插入评论 | The boss, **of course**, sent it. |

### 前置知识:4 种"砖块"

砖块越大,能塞的信息越多。**长难句的本质就是用了大砖块**(从句作主语 / 非谓语作定语等)。

```
单词  →  短语  →  非谓语动词  →  从句
(最小)                          (最大)
```

### 前置知识:二维矩阵(核心图)

|  | 主语位 | 宾语位 | 定语位 | 状语位 | 表语位 |
|---|---|---|---|---|---|
| **单词** | The boss | an email | new | quickly | urgent |
| **短语** | The new boss | an urgent email | from HQ | in a hurry | out of the loop |
| **非谓语** | **Sending** emails is urgent. | He wants **to send** it. | The email **sent yesterday**... | He stayed late **to send** it. | His job is **to send** the email. |
| **从句** | **Whoever was in charge** sent it. | He sent **what he had drafted**. | The boss **who had just arrived** sent it. | He sent it **when the deadline came**. | The truth is **that the boss sent an email**. |

### 4 大类从句速记口诀

> **从句不神秘 = 用一句话顶替一个词。**
>
> - 哪里能放**名词** → 哪里就能放**名词性从句**(主语 / 宾语 / 表语 / 同位语 4 种位置)
> - 哪里能放**形容词** → 哪里就能放**定语从句**
> - 哪里能放**副词** → 哪里就能放**状语从句**

### 第三步:做视译

**目标**:用 [[顺句驱动]] 原则,按意群一段段翻译成地道中文。

#### (a) 划分意群

**成分识别提供候选边界,按意思的完整性决定在哪里落刀**。意群边界通常落在:
- 从句开头(that / who / which / when 等)
- 介词短语开头
- 长定语前后
- 插入语前后

#### (b) 顺句驱动

按原句顺序,**一个意群一个意群翻译**,加连接词(那个 / 就是 / 而且 / 也就是说 / —— 等)形成地道中文。

详见 [[顺句驱动]]。

### 实战例句拆解(参考)

> 完整的三步走实战演示,详见 `01-业务情况/01-繁星相关/快闪训练营_B级别_5月_长难句解析/02-上课实战参考.md`,内含 5 个完整长难句的三步走完整拆解(讲师手册)。

举一个简短示例:

**原句**:
> The knowledge that you are about to inflict some measure of suffering on someone you may care a lot about obviously makes actually doing it very hard.

**第一步:通读** — 大意:意识到要给在乎的人造成痛苦,会让人下不了手。

**第二步:分析成分**
- **母句**:The knowledge makes actually doing it very hard.(主谓宾宾补 = SVOC)
- **从句**:
  - 同位语从句:that you are about to inflict some measure of suffering on someone...(修饰 knowledge)
  - 定语从句:(whom) you may care a lot about(嵌在同位语从句里,修饰 someone)

**第三步:做视译**
- 切意群:The knowledge / that you are about to inflict some measure of suffering / on someone you may care a lot about / obviously makes actually doing it very hard.
- 顺句驱动:**那种意识 —— 你即将造成某种程度的痛苦,对象还是你可能非常在乎的人 —— 这种意识显然让你"真正下手"变得无比困难。**

## CEFR 级别 × 语法分类(大致方向)

> 不同 CEFR 级别学员要学的语法重点。**这里只给大方向,具体语法细节让 AI 查即可,不在 wiki 里展开。**

| CEFR | 语法学习重点 |
|---|---|
| **A1-A2(零基础-入门)** | 五大基本句型 / 主谓宾 / 一般现在/过去/将来 / 基础疑问句和否定句 / 简单介词 |
| **B1(中级)** | 三大句子类型 / 进行时与完成时 / 简单从句(宾语从句、定语从句基础) / 连词和副词 |
| **B2(中高级)** | 完整从句体系(名词性 / 定语 / 状语) / 非谓语动词(动名词、不定式、分词) / 完成进行时 / 被动语态(专项学) |
| **C1(高级)** | 虚拟语气(当固定搭配背) / 倒装结构(当固定搭配背) / 复杂从句嵌套 / 长难句分析三步走 |
| **C2(精通)** | 强调结构 / 修辞性语法(对仗、平行结构) / 高级长难句解析 |

**学习哲学贯穿所有级别**:
- **当固定搭配背的部分**:在 [[中英语块-两个维度|语料储备]] 中附带学到,不单独投入
- **专项学的部分**:只针对"附带学不到"的内容(时态 / 成分 / 从句 / 被动语态)

## 完整理论结构(原"职场实战语法 · 理论篇"大纲保留)

```
1. 基础篇 ── 句子构成
   a. 五大基本句型
   b. 三大句子类型
   c. 连词 & 副词

2. 进阶篇 ── 从句类型
   a. 找成分
   b. 学从句(名词性 / 定语 / 状语)

3. 补充内容 ── 特殊结构
   a. 虚拟语气           ← 当固定搭配背
   b. 倒装结构           ← 当固定搭配背
   c. 强调结构           ← 当固定搭配背
   d. 被动语态           ← 专项学,不是当搭配背

4. 补充内容 ── 时态     ← 专项学
```

## 实战导向 ⭐

**核心心法**:**先实战,再总结规律**——不要倒过来。

### 为什么实战导向

- 语法书系统讲解 + 学员系统学习 → **大量内容学了用不上**
- 学员实战出错 → 老师 / 自己反思 → 缺啥补啥 → **学到的都用得上**

### 三单 / 单复数等碎规律

**实战中总结规律,不要先学规律再实战**。

- 错误路径:背"主语第三人称单数,动词加 s" → 记住规律但实战常忘
- 正确路径:**实战中说错** → 听者纠正或自己感觉不对 → 自己总结规律 → **下次记得牢**

## 跟语料储备的关系

### 很多固定搭配里**暗含语法**

学员积累固定搭配的过程中,**自然就把语法学了一部分**:

- *take time out of one's busy schedule* —— 学到了 *take time out of...* 这个 phrasal verb 结构
- *No sooner had I left than...* —— 学到了倒装的实际用法

**这就是为什么"大部分语法可以当固定搭配背"** —— 它们已经融在语料储备的训练里了。

### 真正剩下要专项学的,只有时态 / 成分 / 从句 / 被动语态

**这就是 Barry 两分法的实质**:
- 已经在语料储备训练里**附带学到的** → 不用再单独投入
- 剩下**附带学不到的** → 才需要专项学习

## 学员常见误区

| 误区 | 真相 |
|---|---|
| "我语法不好,英语肯定不好" | 大部分语法已经在语料储备训练里附带学了。真正缺的可能只是时态 / 成分 |
| "我系统过一遍语法书就能讲好英语了" | 完全错。不实战的语法 = 死知识 |
| "虚拟语气太复杂,我得专门学" | 不必。**当 *If I were... I would...* 整体搭配背**就行 |
| "被动语态记几个例句就行" | ⚠️ **被动语态是要专项学的**——它涉及结构变换,不是固定搭配 |
| "我语法不好但视译能练" | ❌ 语法不到位 → 意群切错 → 顺句驱动跑不起来。**先补成分识别** |
