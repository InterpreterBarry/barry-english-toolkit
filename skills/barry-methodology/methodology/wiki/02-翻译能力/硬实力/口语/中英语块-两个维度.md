# 中英语块 · 两个维度(单词 / 固定搭配+句型)

> **「中英语块 + 视译」这套统一方法里,"中英语块"这条腿装什么料**——就是这两个维度:**①单词 ②固定搭配 + 句型**。两个维度都要从 [[被动词汇vs主动词汇|被动变主动]],同时进行,不是先后顺序。

## 板块定位 ⭐

口语方法论的唯一统一方法是 [[中英语块加视译|中英语块 + 视译]],它分两条腿:

```
中英语块 + 视译(统一方法)
├── 中英语块 ── 练什么 / 储备的内容  ← 本页讲这个
│     ├── 单词维度
│     └── 固定搭配 + 句型维度
└── 视译 ──── 怎么练(过程 / 标准 / 结果)
```

**本页 = "中英语块"这条腿的内容拆解**。为什么是两个维度而不是三个?因为**固定搭配和句型,关注的都是"背后的意思",处理方式完全一样**,合并成一维;而**单词维度有自己独特的处理方式**(要区分本质含义 vs 字面含义),单列。

> 历史:本页原名"训练目标-三个维度"→"训练目标-两个维度"→"中英语块-两个维度"。最终定位:这两个维度是**中英语块的构成**,从属于"中英语块 + 视译"统一方法,不再是跟方法并列的独立"训练目标"。

## 维度一:单词

### 底线:单词必须背

没储备就没输出。但**怎么背**有讲究。

### 三个关键词(怎么背)

| 关键词 | 含义 |
|---|---|
| **上下文(context)** | 配 1-2 个例句,不背孤立单词 |
| **按音节背诵(syllable)** | 不背单字母 |
| **重复记忆** | 系统性复习,符合艾宾浩斯曲线(公共知识,不展开) |

详见 [[词汇学习-两种方式]] / [[间隔重复-Spaced-Repetition]]。

### 核心:关注「本质含义」⭐

单词维度的关键不在于"背了多少",而在于**背的时候有没有抓住这个词的「本质含义」**。

- 有些词**字面含义 = 本质含义** → 只记字面即可(如 *one-on-one* 一对一沟通、*all-hands* 全员大会)
- 有些词**字面含义 ≠ 本质含义** → 必须更多关注本质含义(如 *fork* 不是叉子,是"叉子形状的分叉")

**核心思路**:掌握**本质含义**之后,遇到同一单词的不同用法时**能自然理解**。

> 完整展开(含 Fork / Bandwidth / Runway / Silo 职场例子 + 判断方法)详见 [[本质含义vs字面含义]]。

### 例外:抽象词

抽象、复杂概念词(如 *accommodate*, *implement*, *facilitate*)**没有画面感**——只能:
1. 记文字含义
2. 通过文字含义**揣测背后的意思**
3. 在大量语境里反复见,慢慢内化

## 维度二:固定搭配 + 句型

### 核心:只抓「背后的意思」⭐

固定搭配和句型是**中英语块**的主体。背它们的时候——**不要盯着对照的汉字本身,而要抓住这个表达 / 句型背后的本质意思**。

> 表面是背中英文字,**本质是把 M(意思)— E(英文形式)— S(声音)三者绑在一起**。做到这点,通过科学训练就能**真正打通 M→S 直连**。

### 固定搭配示例

| 中文 | 英文 |
|---|---|
| 百忙之中 | take time out of your busy schedule |
| 提出观点 | put forward an idea |
| 推进入职流程 | move forward with the onboarding process |
| 跳出舒适区 | step out of your comfort zone |

> **背到能脱口而出。** 介词大量出现在固定搭配里,详见 [[介词画面感]]——但不要太纠结,核心是能脱口而出。

### 句型示例

| 中文 | 英文 |
|---|---|
| 相比于 A 我更喜欢 B | I prefer B to A |
| 只是因为...并不意味着 | Just because... doesn't mean... |
| 一边...一边... | while... |

### 特殊句型当固定搭配记 ⭐

英文里有些"语法书会单独讲的"特殊句型,**当固定搭配整体记就行**——不要纠结每个词为什么这样排:

- *No sooner had I... than...*(刚...就...)
- *I would rather do A than B*(我宁愿做 A 而不是 B)
- *It is not until... that...*(直到...才...)

**为什么不要纠结语法**:这些句型用语法解释会很复杂(完成时倒装、从句嵌套等),**直接整体当搭配记**反而效率高。

详见 [[语块概念-Meaning-Unit]] / [[语块优先原则]]。

## 两个维度的关系

### 不是先后顺序,是同时进行

学员经常以为"先背完单词,再背搭配句型"——**错**。两个维度**同时进行**,在每次 [[中英语块加视译|视译]] 训练里**都同时被激活**。

### 视译把两个维度一起练活 ⭐

中英语块储备好了,要靠 [[中英语块加视译|视译]] 才能从被动变主动:

```
中英语块(两个维度的料)
   ↓ 视译(过程 / 标准 / 结果)
├── 单词:在拼句子过程中反复调用 → 主动化
└── 固定搭配 + 句型:整体进入、整体输出,作为框架反复使用 → 主动化
```

**不需要为单词单独练一种方法、语块单独练一种方法**——视译训练**天然耦合**了两个维度。

## 两个维度上,什么样的语料值得背

### 单词层

- 高频核心词汇(职场场景)
- 你工作中**反复遇到**的词

### 固定搭配 + 句型层

- **必须与你的工作内容相关**——背别的行业的搭配,大概率用不到
- 高频职场搭配(如 take time, follow up, push back...)
- 表达**观点 / 立场 / 比较 / 因果**这类高频职场逻辑的句型,能在演讲 / 汇报里**直接派上用场**
