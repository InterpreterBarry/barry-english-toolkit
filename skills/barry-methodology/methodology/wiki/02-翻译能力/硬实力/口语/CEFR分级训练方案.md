# CEFR 分级训练方案

> 不同级别的学员用不同的训练方案。**A 级打地基、B 级大量视译、C 级高标准演讲**——不分级硬练,容易要么难到放弃、要么简单到没增量。

## 核心定位

- CEFR(Common European Framework of Reference for Languages)是欧洲共同语言参考标准,A1-C2 六级
- Barry 方法论取 **A / B / C 三档**,作为训练方案分水岭
- 分级决定**用哪种训练**、**多大强度**、**多少脱壳**,不是决定"能不能训练"

## WHAT:三级方案

| 级别 | 训练重点 |
|---|---|
| **A 级别** | 把固定搭配和语块背熟 |
| **B 级别** | 大量视译训练 + 英文演讲 / 口语角 |
| **C 级别** | 高标准演讲(准备 3 分钟演讲,框架 + 细节完善) |

## WHY:为什么要分级

**一刀切的问题**:
- A 级学员没有语块储备,直接上视译 → 只能憋出半句半句的 broken English,挫败感极强
- C 级学员继续背语块 → 储备过剩,输出却没见长,卡在"认识但不用"
- B 级学员跳过视译直接冲演讲 → 框架撑得起来,但每句话都在现编,不流畅

**匹配逻辑**:
- **A 级**:储备 < 表达需求 → 先补储备
- **B 级**:储备 ≈ 表达需求 → 练调用
- **C 级**:储备 > 日常表达需求 → 练高密度场景(演讲)

## HOW:每一级怎么练

### A 级 — 补砖

- 背职场固定搭配、常见语块
- 每个语块配上例句(不背孤立短语)
- **KISS 约束**([[KISS原则]]):能用简单句就别上复杂句,重点是把语块用对
- 搭配 [[词汇学习-两种方式]]:intentional learning 为主,incidental learning 为辅

### B 级 — 练调用

- [[视译训练-Sight-Interpreting|视译]] 每天半小时,一小段
- [[复述-Retelling]]:3 分钟材料听完后按框架复述
- 英文演讲 / 口语角:实战输出

### C 级 — 练高密度输出

- 准备 3 分钟演讲(框架 + 细节都要完善)
- 重点是**结构**(逻辑思维板块)+ **语言密度**(词汇、句式多样性)
- [[Mini-Presentation训练方法论]] 的完整流程
- [[4-3-2-技巧]] 上逐轮提速

## 与 [[KISS原则]] 的交互

KISS 在每一级都适用,但**容忍度不同**:

| 级别 | KISS 执行强度 |
|---|---|
| A | 非常严格 — 能用 5 词表达不上 10 词 |
| B | 一般严格 — 允许在输出里尝试复杂表达,但出错了回归简单 |
| C | 相对宽松 — 演讲场景里的修辞、节奏、句式多样性是加分项,但仍不允许绕弯子 |

## 如何判断自己属于哪一级

- 用 [[学员定级三维评判]] 做综合判断
- 三维:词汇储备 / 逻辑组织 / 听力理解
- 不要只看一个维度——有人词汇厚但逻辑乱,有人逻辑清但词汇薄,训练方案完全不同
