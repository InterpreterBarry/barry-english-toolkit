# 词汇学习的两种方式

> **附带词汇学习**(Incidental)vs **有意词汇学习**(Intentional)。两种都有价值,但**有意学习结合语境接触**才是最快路径。

## 核心定位

- 这是 Barry 方法论对"背单词"这件事的**方法论分类**
- 不是"哪个对哪个错",是"什么时候用哪个"
- 对**已有基础的中国职场人**(方法论主要受众):有意学习为主 + 附带学习兜底

## WHAT:两种方式

### 附带词汇学习(Incidental Vocabulary Learning)

- **机制**:在阅读、听力、对话等**语境接触**中自然学到新词
- **特点**:不刻意、慢、但自然、建立语感
- **效果**:有助于建立**被动词汇**(看得懂听得懂)

### 有意词汇学习(Intentional Vocabulary Learning)

- **机制**:有目的、有意识地**专门**学习和记忆单词(背单词书、做闪卡)
- **特点**:刻意、快、能快速扩张词汇量
- **效果**:更快建立**主动词汇**(能调用出来)

## WHY:为什么方法论选有意为主

**研究结论**:对成人二语学习者,intentional + 语境接触的组合效果 > 纯 incidental。原因:

1. **单纯阅读附带学习**:一个生词要在不同语境中遇到 **8-12 次**才能稳定记住。对于学员水平匹配的材料,这种暴露频率在短时间内很难达成
2. **有意学习 + 语境**:通过有意识背诵建立首次记忆,再通过语境接触**固化 + 拓展**,效率高得多
3. **Nation 的 Four Strands 建议**:[[Language-focused-Learning]] 占 25%(有意学习在此),[[Meaning-focused-Input]] 占 25%(附带学习在此)——两条腿走路

## HOW:两种方式怎么配合

### 有意学习三个核心技巧

1. **通过音节背**,不要背单个字母
2. **在语境(Context)中背**——同一个词配 1-2 个例句
3. **系统性复习**(而非单纯重复)——走 [[间隔重复-Spaced-Repetition]]

### 附带学习的落地

- 每天安排 [[Meaning-focused-Input|可理解输入]](泛读 / 泛听)
- 材料难度匹配 **95-98% 词汇已知**,生词控制在 1-2%
- 不查所有生词,只查**反复出现且影响理解**的

### 分工

| 场景 | 用哪种方式 |
|---|---|
| 准备考试 / 快速补基础 | 有意学习为主 |
| 长期语感维持 | 附带学习为主 |
| 背新领域专业术语 | 有意学习 |
| 磨语感、练搭配、体会一词多义 | 附带学习 |

## 什么样的词适合直接背

- **基础单词**:零基础阶段直接背,没有语境也能背(先有砖才能砌墙)
- **职场语块**:不单独背一个词,背 [[语块概念-Meaning-Unit|整个语块]](详见 [[语块优先原则]])
- **低频但重要的专业术语**:出现不多,但遇到必须认识,直接背

## 什么样的词靠附带学习更好

- **一词多义**:apple 你不是背 10 个意思,是读得多了自然遇到
- **搭配灵活性**:push back 在不同语境有不同感觉,背不全,读得多才体会得深
- **文化语感**:什么话适合什么场合说,这个书里背不出来
