# 输入输出并行原则

> **学了的东西要拿到实战里用,用的过程中也要持续学。** 中文里说"输入输出并行"——这两个词的真实含义,跟 Paul Nation 英文学术语境用的 "input/output" **不是一一对应**。中文学员语境下:**输入 = LFL(刻意学习语言形式)/ 输出 = MFI + MFO(实战中的听说读写)**。

## 板块定位 ⭐

[[0-翻译能力总览|02-翻译能力]] / [[0-硬实力总览|硬实力]] / 宏观维度 · Four Strands / **输入输出并行原则**。

## 措辞精确化 ⚠️(重要)

**中文措辞 → Four Strands 真实映射**:

| 中文措辞 | 在 Four Strands 里对应 | 含义 |
|---|---|---|
| **输入** | **LFL**(Language-focused Learning) | **刻意学习语言形式** —— 背词块、学语法、词汇深度拓展、表达精讲 |
| **输出** | **MFI + MFO** | **实战中的沟通** —— 在场景中听懂、读懂、说出来、写出来 |

### 为什么会有这个错位

| 语境 | "输入 / 输出" 的切分维度 |
|---|---|
| **英文学术语境**(Paul Nation) | 按"语言信号方向":Input = 接收信号(听/读)/ Output = 产出信号(说/写) |
| **中文学员语境**(Barry 体系) | 按"学习动作":输入 = 学到肚子里(刻意学)/ 输出 = 拿出来用(实战) |

**所以中文学员说"输入输出要并行"**,实质讲的是 **LFL ↔ (MFI + MFO)** 这两组之间的并行——**学了的东西要拿到实战里用,用的过程中也要持续学新的**。

> ⚠️ 不要错把"输入"理解成 Paul Nation 的 MFI(听/读)。MFI 在中文学员语境下属于"输出"——属于"实战应用"的一部分。

## 核心观点 / WHAT

**两条轨道必须同步推进**:

| 轨道 | = Four Strands 哪条线 | 有效形式 |
|---|---|---|
| **输入轨道**(刻意学习语言形式) | LFL | 词块精讲、语法专项、表达深度拓展、Anki 背词 |
| **输出轨道**(实战中的沟通应用) | MFI + MFO | 听 BBC / 看美剧 / 读原版书 / 口语 / 写作 / 模拟会议 |

### 两种失衡的典型症状

| 失衡类型 | 表现 | 典型人群 |
|---|---|---|
| **只学不用**(只 LFL 不 MFI/MFO) | 积累了 5000 个"认识但用不出来"的词 | 中国职场人典型症状(背了一堆没场景用) |
| **只用不学**(只 MFI/MFO 不 LFL) | 沟通流畅但**水平停滞**,所有 output 都在重复浅词块 | 长期国外生活但没系统学习的学员 |

## WHY:为什么必须并行

### 只学不用 → 被动词汇陷阱

学员积累了大量被动词汇——**看到 / 听到能认识,但用不出来**。详见 [[被动词汇vs主动词汇]]。

### 只用不学 → 水平天花板

没有持续注入新的语言形式 → 实战只能复用已有的浅词块 → **流畅但永远停在原地**。

### 双轨闭环才有提升

```
新学的语料(LFL 输入)
        ↓
进入实战(MFI/MFO 输出)
        ↓
暴露问题(用得不准 / 没记牢 / 调不出来)
        ↓
倒回来补(再次 LFL 输入,有的放矢)
        ↓
重新进入实战
```

**循环闭环**才是真正的"并行"——不是机械地"50% 学 + 50% 用",而是**两条轨道互相喂料**。

## HOW:核心推荐方法 = 视译

**视译统一归 LFL**(横跨 LFL/MFO,为简化归 LFL):
- **LFL 层面**:学新的中英语块(刻意学习)
- **MFO 层面**:把语块用出来(实战拼句子)——严格说也算练产出,但为简化理解统一归到 LFL

**一个动作同时跑两条轨道**,性价比最高。详见 [[视译训练-Sight-Interpreting]] 和 [[中英语块加视译]]。

## 反例:典型失衡学员

**小王的数据**:
- 美剧 100 集 + TED 50 个 + Anki 背词 5000 → **MFI 和 LFL 都做了一些**
- 但**几乎不做 MFO**(不开口说、不写作)
- 结果:能听懂 BBC,一张口还是 "嗯... this... that... maybe..."

**诊断**:**输出端 MFO 严重缺位**——被动词汇上万,主动词汇不到 200。

**药方**:每天 15 分钟**视译 + 限时 Retelling**(视译归 LFL〔横跨 LFL/MFO,为简化归 LFL〕;限时 Retelling = MFO + FD),3 个月可见变化。

## 学员定制方案的应用

详见 [[Four-Strands-训练方法工具箱]]——三档统一 1/3 : 1/3 : 1/3(MFI / MFO / LFL 三线均分),级别差异只体现在素材难度和 FD 的有无,不体现在三线比例上。
