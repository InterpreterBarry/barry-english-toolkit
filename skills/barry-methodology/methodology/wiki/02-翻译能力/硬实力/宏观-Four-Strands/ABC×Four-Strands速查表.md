# ABC × Four Strands 速查表

> 一图看懂：CEFR 三档（A/B/C）在 Four Strands 框架下分别怎么练。压缩自 [[Four-Strands-训练方法工具箱]]（方法库）+ [[CEFR分级训练方案]]（应用层）。
> 一句话记忆：**A 级打地基（补储备）· B 级练调用（大量视译）· C 级练高标准输出（高标准演讲）**。

---

## 配比总表（三足鼎立 + FD 嵌入）

| Strand | 核心 purpose | A 级 | B 级 | C 级 |
|---|---|---|---|---|
| **MFI** 意义聚焦输入 | 理解意思，焦点在内容（非刻意学形式）| 1/3 | 1/3 | 1/3 |
| **MFO** 意义聚焦输出 | 传达信息，焦点在内容（非刻意学形式）| 1/3 | 1/3 | 1/3 |
| **LFL** 形式聚焦学习 | 刻意关注形式 / 间隔重复 | 1/3 | 1/3 | 1/3 |
| **FD** 流利度发展 | 把已知内容用得更快 | ❌ 不做（A 级先打基础，B 级以上再加 FD，理由见 [[Four-Strands-与Barry方法论的融合]]）| ⚙️ 嵌入加压 | 🔥 强制 4-3-2 |

> ⚠️ **核心警示**：中国学员普遍过度倾向 LFL，塌掉 MFI / MFO。配方案时必须守住三足鼎立。
> **FD 不单独占配比**——它是融入复述(听材料=MFI;首轮产出=MFO;限时加压轮=FD)+ MFO Mini-Presentation 的时间压迫策略。

## 各训练方法 × ABC（按 Strand 分组）

| Strand | 训练方法 | A 级 | B 级 | C 级 |
|---|---|---|---|---|
| **MFI** | 分级阅读 | 短 + 易（AI 生成职场读物）| 中等 | 长 + 难 |
| **MFI** | 分级听力 + 复述 | AI 推 VOA / BBC Learning English | 自找 YouTube / TED | 自找 YouTube / TED |
| **MFO** | 对话 / 口语角（AI 当外教）| ✓ 主题简单 | ✓ 主题中等 | ✓ 主题难 |
| **MFO** | Mini Presentation | ✓ 基础 | ✓ + 时间压迫 | ✓ + 强化 4-3-2 |
| **MFO** | 写作（按需）| ✓ | ✓ | ✓ |
| **LFL** | 词块 / Anki 单词卡 | 几乎全收 | 跳基础高频，收次高频 | 只收地道高阶 / 学术词 |
| **LFL** | 中→英翻译（视译归 LFL，横跨 LFL/MFO 为简化归 LFL）| **笔译** | **视译** | **视译** |
| **LFL** | 英→中（听写 + 复述）| 重打听写地基 | 重打听写地基 | 综合复述 |
| **LFL** | 语法（融入精读）| 简单句 / 并列句 | 各类从句 + 插入语 | 最复杂句式 |
| **LFL** | 精读（翻译三步走）| ✓ | ✓ | ✓ |
| **LFL** | 发音 | 筛严重问题 → 1V1 | 筛严重问题 → 1V1 | 筛严重问题 → 1V1 |
| **FD** | 4-3-2 提速 | ❌ 不需要 | 适当加入 | 强化 |

## 一级一句话（应用层逻辑）

| 级别 | 储备 vs 表达需求 | 训练主轴 | KISS 执行强度 |
|---|---|---|---|
| **A 级** | 储备 < 需求 → **补储备** | 背职场固定搭配 / 语块（配例句）| 非常严格：能 5 词不上 10 词 |
| **B 级** | 储备 ≈ 需求 → **练调用** | 每天半小时视译 + 复述 + 口语角 | 一般严格：可试复杂，错了回归简单 |
| **C 级** | 储备 > 需求 → **练高标准** | 3 分钟高标准演讲（结构 + 词汇句式多样性）| 相对宽松：修辞/句式多样是加分项 |

---

## Cross-Cutting 总原则（贯穿三级）

1. **词块积累必选**：所有 AI 反馈都要产出 4 列词表（英文｜音标｜中文｜例句）→ 同步 Anki 网页
2. **筛选门槛随级别单调递增**：避免 C1 词库塞满 take / make 这种浪费时间的词
3. **尽量每天练不中断**：每周最多休 1-2 天，疲劳可轻量化不停训
4. **档案不主动催重测**：学员主动汇报"换工作/进步/换目标"时才建议重走 onboarding

---

## AI 能力边界（出方案时必须标给学员）

- ⚠️ **英→中听力片段**：AI 无法生成 → 学员自找素材
- ⚠️ **发音纠音**：AI 做不到 → 建议 1V1 老师
- ⚠️ **长时语音对话**：宕机时用语音转文字 APP（Typeless / 闪电说）backup
