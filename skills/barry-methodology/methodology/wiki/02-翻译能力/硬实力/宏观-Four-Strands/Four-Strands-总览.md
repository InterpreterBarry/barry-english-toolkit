# Four Strands 总览

> Paul Nation (2007) 提出的语言学习总框架:所有学习活动可归入四条线,理想状态下**各占 25% 的总学习时间**。

## 核心框架

Nation 的核心主张:一门健康的语言课程(课内 + 课外合算)应该在四条线上**大致均衡分布**。

| Strand | 注意力焦点 | 典型活动 |
|---|---|---|
| [[Meaning-focused-Input]] | 意义(接收) | 泛读、听故事、看剧 |
| [[Meaning-focused-Output]] | 意义(产出) | 对话、演讲、写作 |
| [[Language-focused-Learning]] | 形式(刻意) | 背词、语法、精讲、翻译 |
| [[Fluency-Development]] | 速度(自动化) | 速读、4/3/2、限时输出 |

## WHAT:两个维度的交叉

四条线来自两个维度的**交叉**,加一条独立维度:

- **注意力焦点**:意义(meaning) vs 形式(form)
- **语言流向**:输入(input) vs 输出(output)
- **独立维度**:Fluency Development——不学新东西,只把已知内容自动化

## WHY:为什么要四条线均衡

**核心信念**:任何一条线单独拉满都会失衡。

- 只做 Language-focused Learning → 认识词但用不出来(中国学习者的典型问题)
- 只做 Meaning-focused Input → 能听懂读懂,但不会说不会写
- 只做 Meaning-focused Output → 表达受限于已知词汇,天花板低
- 缺 Fluency Development → 永远在"组装模式",说不流畅

Nation 的 25/25/25/25 配比是**理想状态下的时间分配建议**——不是每堂课 25%,而是**学习者整体学习时间**的均衡。

## ⭐ Barry 方法论里的实际配置(关键,别只看 Nation 原版)

Barry 方法论里 Four Strands **不是 Nation 原版的 25/25/25/25**,而是按中国职场学员实际:

```
MFI : MFO : LFL = 1/3 : 1/3 : 1/3   (三足鼎立)
FD                = 不独立占配比     (作为 Time Pressure 嵌入策略)
```

**FD 作为时间压迫策略嵌入复述和 Mini-Pre**——复述:听材料 = MFI,首轮复述 = MFO,限时加压重复轮 = FD;Mini-Pre 同理(产出 = MFO,4-3-2 加压 = FD):

| 级别 | FD 在 MFI/MFO 中的应用 |
|---|---|
| **A 级** | **完全不考虑**(基础阶段做不到,要求太高)|
| **B 级** | 适当加入(AI 在复述 / Mini-Pre 训练时给"再压缩 X 秒"指令)|
| **C 级** | **必含**(每次复述 / Mini-Pre 都强制走 4-3-2 流程,时间逐轮压缩)|

**核心约束**:**没时间压力 ≠ Fluency**(Nation 原话,见 [[Fluency-Development]] 条件 3)。Retelling / Mini-Pre 第一遍不限时追求完整,第二遍起时间减半 → 同一任务自然覆盖 MFO + FD 两条线。

详见 [[Four-Strands-与Barry方法论的融合]] / [[Four-Strands-训练方法工具箱]] / [[4-3-2-技巧]]。

## HOW:关键实操原则

### 1. 看目的,不看形式

同一个活动落在哪条线,取决于学习者**当下的 purpose/focus**,不是活动形式本身:

- 听一段新闻听懂大意 → Meaning-focused Input
- 听同一段新闻专门练数字听辨 → Language-focused Learning
- 酒吧里朋友第二遍讲同一个故事 → Fluency Development

### 2. 三条 meaning 线 vs 一条 language 线

另一种读法:三条 meaning-focused 线(input + output + fluency)占 75%,language-focused 只占 25%,**比例是 3 : 1**。这提醒学习者:**大部分时间应该花在"用语言"上,而不是"学语言"上**。

### 3. Barry 方法论的缺口

繁星体系目前在 Language-focused Learning 和 Meaning-focused Output 上很强,但 Meaning-focused Input 几乎为零、Fluency Development 缺显性时间压力。详细映射见 [[Four-Strands-与Barry方法论的融合]]。
