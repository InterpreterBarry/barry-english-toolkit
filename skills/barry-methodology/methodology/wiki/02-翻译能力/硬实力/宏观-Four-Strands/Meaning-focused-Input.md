# Meaning-focused Input

> Four Strands 的第一条线:**通过听和读学习,注意力在理解意思**。大量可理解输入是语言积累的底座。

## WHAT:意义焦点的接收性学习

Meaning-focused Input(意义焦点输入)指学习者**为理解内容本身**而去听或读——不是为了学词、不是为了学语法,就是**想知道故事/信息/观点**。

典型活动:泛读、听故事、看英剧、shared reading、听播客。

**关键判断标准不是活动形式,而是学习者的 purpose**(详见 [[Four-Strands-总览]] 里「看目的,不看形式」)。同一个听力材料,**听懂意思 = Meaning-focused Input;停下来查词学搭配 = Language-focused Learning**。

## WHY:为什么必须有这条线

Nation 认为这条线应占学习时间的 **25%**(含课外)。原因:

- **附带学习(incidental learning)效率低但量大累积巨大**:每碰到一个词只记住一点点,但几百小时累积下来,词汇的多义性、搭配灵活性、语感都是这样建立的
- **这些东西靠背语块补不全**:语块精讲给你"正确答案",但只有大量真实输入才能让你**见过足够多的句式**,知道什么时候用什么
- **Barry 体系的明显缺口**:繁星的听力都是"为了学习",学员几乎没有纯粹享受内容的输入量

## HOW:五个前提条件(Nation)

1. **大部分内容学习者已经熟悉**(主题、背景、情境)
2. **学习者对内容感兴趣**(否则坚持不下来)
3. **95-98% 词汇已知**——每 100 词只有 1-2 个生词
4. **能通过上下文猜测生词**
5. **大量输入**——量要足够大才有累积效应(Nation 给的参考量级:约每两周读完一本 graded reader,是举例不是硬指标)

第 3 条最容易被忽视:**材料必须难度匹配**。太难(低于 95% 已知)→ 满足不了"可理解",会变成 Language-focused Learning;太简单 → 没有增量。

## HOW:Local Listening 也是 Meaning-focused

"听具体信息"(如火车几点开)同样属于 Meaning-focused Input,而非 Language-focused:

- Global listening(听大意) = Meaning-focused Input
- Local listening(听具体信息) = **仍然是 Meaning-focused Input**
- 只有当目的变成"刻意练某个听辨策略"时才变成 Language-focused

详见 [[Global-vs-Local-Listening]]。

## HOW:Barry 体系的补救路径

1. **Graded readers + 配套音频**:A2 / B1 级别分级读物
2. **语言教师播客**:教师自然分级,天然 comprehensible
3. **改造现有听力素材**:加长 clip,去掉语言训练任务,改为"听完复述大意"
4. **Mini Presentation 互听**:听别人的产出对 listener 就是 input
5. **课外引导 > 课内占满**:社群课内做 Language-focused 没问题,但**必须强调学员课外补这条线**

## 示例

**不是 Meaning-focused Input**:精听一段 BBC 新闻,每个生词停下来查词典,做听写。
**是 Meaning-focused Input**:同一段新闻,目标是听懂今天发生了什么,生词猜一下就过去,不查词典。
