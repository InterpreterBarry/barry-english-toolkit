# Phase 1.3:对齐标准(怎样算解决了?)

> 定义清楚之后,确定**衡量标准**:什么是好、什么是不好、怎样算解决了、怎样算完成了。

## 核心

**标准不同,结论完全不同。** 跳过 Phase 1.3 就会出现:A 觉得已经解决,B 觉得还差很远——两人没对错,是用的尺子不同。

## WHAT:Phase 1.3 要做的一件事

问题定义好之后,追问:**"我们用什么标准来判断这件事?"**

## WHY:没有标准 → 没有结论

没有对齐的衡量尺,讨论会停在一个"看起来像结论但其实没人能执行"的地方——因为执行者也不知道做到什么程度就算做完了。

## 示例:"要不要裁这个人?"

```
标准 = 绩效          ── 近 3 个月 KPI 未达标,裁
标准 = 态度          ── 工作态度端正,留
标准 = 潜力          ── 年轻有培养空间,留
标准 = 团队配合度    ── 同事反馈差,裁
```

**四个标准,四个结论。** 这时候必须先拉齐:"在这家公司、这个时间点,我们最看重哪条?"

## HOW:三步对齐

1. **列候选标准**:绩效、态度、潜力、配合度……每个维度都摆出来。
2. **选主次**:这场讨论里,哪个是**主标准**(必过),哪些是**辅助标准**(加减分)?
3. **给权重或底线**:主标准达到什么程度才算通过?辅助标准可以打多少分折算?

## Phase 1 三个子动作的意义

- Phase 1.1:**这是真问题吗?**(问题本身)
- Phase 1.2:**我们说的是同一件事吗?**(词的口径)
- Phase 1.3:**怎样算搞定了?**(尺子)

**三个都对齐之后,才有资格进入 Phase 2 拆解。** 没拉齐就开始搭金字塔,每一块砖都可能是错的。
