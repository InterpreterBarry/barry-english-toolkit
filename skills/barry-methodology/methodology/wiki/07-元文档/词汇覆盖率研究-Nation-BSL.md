# 词汇覆盖率研究 · Nation / Hsu & BSL

> 词汇覆盖率研究(Nation 范式 + Hsu 商务数据)+ BSL(Business Service List)对标 + Barry 自编教材词汇统计。**职场英语的瓶颈不是词汇量不够,是认识的词用不出来**——所以语块和输出训练才是关键。

## WHAT:词汇覆盖率研究(Nation 范式 + Hsu 商务数据)

下表的商务覆盖率数字出自 **Hsu (2011)** 对商务教材 / 论文语料的研究;"多少词族 = 多少覆盖率"这一范式与通用语料阈值的奠基是 **Nation (2006)**:

| 词汇量(词族) | 覆盖率 | 对应水平 |
|---|---|---|
| **3,500 词族** | 商务教材 95% | 基本能看懂 |
| **5,000 词族** | 商务教材 98% / 商务论文 95% | 流畅阅读 |
| **8,000 词族** | 商务论文 98% | 完全无障碍 |

> **词族(word family)** = 一个词根 + 所有派生词(manage / manager / management / manageable 算一个词族)

## WHY:95% vs 98% 的区别

这 3% 的差距,**体感差异巨大**:

- **95%**:每 20 个词有 1 个不认识 → 勉强理解大意,读得累
- **98%**:每 50 个词有 1 个不认识 → 流畅阅读,靠上下文可猜

职场英语设计目标:**达到 98% 覆盖 = 流畅阅读**。对应大约 5,000 词族(商务教材)或 8,000 词族(商务论文)。

### 不同领域差异

- **运营管理类**最简单(3,000 词族 → 95%)
- **商业战略类**最难(5,000 词族 → 95%)

## WHAT:BSL(Business Service List)对标

BSL(Browne & Culligan 2015)基于 **6,400 万词商务语料库**,包含 **1,744 个商务专有词族**,是目前最权威的商务英语专用词表。

### Barry 教材覆盖情况(2026-03-07 统计)

**Barry 的两本自编教材**:《职场核心词汇精讲》Part 1(34 课)+《职场英语高频词汇》(5 单元)。

- BSL 1,744 词族中,Barry 两本书**主讲覆盖约 91 个**(例句另触及一批)
- 未覆盖的约 1,653 个,按领域客观分布(BSL 分析数据 `bsl_analysis_results.json`):**窄领域专业术语**——金融/会计(162)+ 经济(54)+ 法律(47)≈ **263 个**,超出通用职场范围、不教;**通用/跨领域**——General(1,323)+ 运营/HR/市场/技术(约 67)≈ **1,390 个**,这才是"通用职场英语"该覆盖的主体
- ⚠️ 所以通用职场词缺口**是上千量级**("仅 37 个"基于过宽的人工术语分类,已废弃)——正好说明值得正向建一张通用职场核心词表(规划见下)

### 可补充的通用职场高频词(举例)

下面这些是 BSL 里偏通用、Barry 教材目前着墨较少、**下一版可以补**的方向(举例,非穷尽):

| 类别 | 词汇 |
|---|---|
| 商务评价 / 描述 | robust, feasible, unprecedented, resilient, proficient |
| 组织 / 治理 | governance, authority, entity, subsidiary, restructure, downsize, redundancy |
| 财务 / 运营 | overhead, turnover, premium, lease, consumption, utility, valuation, default |
| 市场 / 商业模式 | niche, subscription, franchise, entrepreneur, sponsorship |
| 趋势 / 变化 | recession, downturn, fluctuation, threshold, deploy |
| 人事 / 管理 | appraisal, credential, trainee, harassment |
| 其他高频 | scrutiny, integrate, spreadsheet |

这些是教材**下一版可以补充**的方向。

## WHY:Barry 教材的差异化优势(231 个 BSL 没有的词)

两本书中有 **231 个词族是 BSL 没有的**,包括:

- **会议实战词**:mute, unmute, sync, townhall, huddle, housekeeping, RSVP
- **项目管理词**:blocker, roadmap, dashboard, tracker, Gantt, RACI, postmortem
- **职场俚语**:headwind, tailwind, low-hanging fruit, move the needle, drop the ball

这些词**太新 / 太口语化**,语料库统计无法捕捉,恰恰是 Barry 教学的**核心竞争力**——Barry 的教材不追求学术覆盖率,追求**真实职场会议 / 协作里用到的语言**。

### 与 Nation 语料库的关系

- 这 400-500 词族中约 **80-85% 落在前 3,500 高频词族内**(第 1,000-3,500 频率段)
- 剩余 15-20% 为更专业的术语(RSU、RACI、BPO、OKR 等),落在 3,500 以外
- **两本书的核心价值不在词族覆盖量**,而在**每个词族配了职场语境 + 搭配用法 + 例句**,即"从认识到会用"的桥梁

## HOW:核心结论

> **职场英语的瓶颈不是词汇量不够,是认识的词用不出来。**

95% / 98% 覆盖的前提是"**认识这些词**",但对中国学习者,大部分常用职场词汇是"**半认识**"——见过,模糊知道什么意思,用起来调不出来。

所以:

- **不要继续扩大词汇量**(已经够用)
- **转向语块积累 + 输出训练**(让被动词变主动词)

详见 [[语块优先原则]] 和 [[语块概念-Meaning-Unit]]。
