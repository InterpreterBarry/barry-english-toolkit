# Delivery 理论锚:Physical / Vocal Image

> Delivery 章节的**底层论证**。**Physical Image → Assumption** + **Vocal Image → Assumption turned to belief**。这是为什么 Delivery 不是"小修小补",而是直接决定听众对你的判断。

## 板块定位

[[0-公众演讲板块总览|公众演讲板块]] / [[0-半即兴演讲总览|半即兴演讲]] / **传达(Delivery)章节** —— 整个 Delivery 章节的开篇理论锚。

## 核心论证 ⭐

```
Physical image  ────→  Assumption(产生假设)
                            ↓
Vocal image     ────→  Assumption turned to belief(假设变为信念)
```

> 来自 Feishu 课件最新版,公众演讲板块 Delivery 章节的核心论证。

## WHAT:两个 Image 各做什么

### Physical Image —— 听众**先**通过你的物理形象**形成假设**

听众进场坐下,**还没听你说话**,已经开始判断:
- 你的**站姿**(自信 vs 局促)
- 你的**衣着**(专业 vs 随意)
- 你的**手势**(开放 vs 紧张)
- 你的**表情**(放松 vs 紧绷)

→ 这些**视觉线索**让听众**对你产生第一波假设**:这人值不值得听?

### Vocal Image —— 听众**通过你的声音**把假设**变成信念**

你开口讲话后,听众听到:
- **音量**(够不够撑得住场?)
- **音调**(单调还是有起伏?)
- **节奏**(从容还是慌乱?)
- **停顿**(有思考还是磕巴?)

→ 这些**声音线索**让听众**确认或推翻**之前的假设。当假设被声音验证时,**就变成了信念**——"这人确实值得听 / 这人确实不行"。

## WHY:为什么这个理论锚重要

### 这就是为什么 Delivery 不是"装饰"

很多人觉得:**内容好就行,Delivery 是次要的**。

但**听众的判断顺序是反过来的**:
```
看你 → 形成假设
听你 → 假设变信念
内容 → 在已有信念下被解读
```

**如果 Physical/Vocal Image 不到位**,听众**还没真正听内容**就已经形成了"这人不行"的信念——之后再好的内容**也只能在这个滤镜下被解读**。

### 反过来也成立

**如果 Physical/Vocal Image 到位**,听众**带着信任来听内容**——同样的内容,**接受度高得多**。

**这就是为什么 Mentality 是底色**:Delivery 的所有技巧都建立在"**听众对你的物理 + 声音形象产生信任**"这个前提上。如果你内心紧张外在装作自信(违反 [[板块核心心态|Congruence]]),Vocal Image 一开口就会**暴露你**——假设变信念的方向就负向了。

## Delivery 章节的 3 个维度(顺序按重要性)

```
State(紧张应对)─→ Voice(声音) ─→ Appearance(外在)
   ↑                ↑                ↑
   理论支撑          理论支撑          理论支撑
   (心态)           (Vocal Image)    (Physical Image)
```

**飞书课件的顺序是 State → Voice → Appearance**(状态先于声音和外在)——因为状态决定了你能否真正发挥 Voice 和 Appearance 的能力。**Voice 排在 Appearance 之前**,理由就在本页核心论证里:Physical Image 只**产生假设**,Vocal Image 才把**假设变成信念**——后者是决定性的那一步,所以声音权重高于外在。
