# 半即兴演讲(Extemporaneous)· 总览

> 公众演讲板块的**第一大类型**(跟即兴并列),职场演讲的**默认形态**——有大纲、有关键词,但不逐字背稿。**两章结构**:Content(说什么:Outline → 标准结构 → Q&A)+ Delivery(怎么说:理论锚 → State → Voice → Appearance)。

## WHAT:为什么"半即兴"是职场默认形态

汇报、路演、培训、正式演讲——绝大多数职场演讲都是**提前可准备、现场不照稿**:

- 照稿(Manuscript)眼神和语调死板,只留给法务声明等高风险场景
- 纯即兴(Impromptu)没准备,是被动应对
- **半即兴 = 提前准备结构 + 现场用自然语言填充**——既有掌控,又有临场感

**学习顺序**:Content 先于 Delivery——先保证"说什么"成立,再打磨"怎么说"。
