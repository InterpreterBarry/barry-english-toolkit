# 即兴演讲 · 心态(Mindset)

> 即兴演讲的**第一步**(优先于结构)。**两条心法**:Dare to be dull(敢于平淡)+ Conversation, not performance(对话,不是表演)。本质是 [[板块核心心态]] 在即兴场景的具体落地。

## 板块定位

[[0-公众演讲板块总览|公众演讲板块]] / [[0-即兴演讲总览|即兴演讲]] / **2.1 心态(Mindset)** —— 即兴演讲两层结构的第一层。

## 核心论证:为什么 Mindset 优先于 Structure ⭐

```
即兴演讲被叫上台
   ↓
心态崩 → 即兴必崩(再多结构也救不了)
   ↓
心态稳 → 即兴有救(套个 5 框架就能讲)
```

**半即兴**有准备、有结构,心态紧也能用结构兜底。
**即兴**没准备,**只剩心态兜底**。

所以即兴演讲的**起手式是 Mindset,不是 Structure**。

## 两条心法

### 1. Dare to be dull(敢于平淡)⭐

> **不需要金句,把话说完整已经比大部分人强。**

学员临场被叫上台,**最容易踩的坑是想"讲精彩"**——结果:
- 想用金句 → 想不出来 → 卡壳
- 想搞个段子 → 没素材 → 尴尬
- 想华丽辞藻 → 切换不出来 → 磕巴

**Dare to be dull 的真正含义**:

接受**自己临场就讲一段平实的话**——
- 不抖机灵
- 不强行制造高潮
- 不假装幽默

**讲清楚 > 讲精彩**。**完整 > 华丽**。

> **大部分听众的标准是"听得懂",不是"被惊艳到"**。能讲清楚一件事,已经超过 80% 的人。

### 2. Conversation, not performance(对话,不是表演)⭐

> **想象你在跟一个朋友聊天,不是在 stage 上表演**。

"表演"心态的问题:
- **自我审视**:我看起来如何?声音稳吗?手放哪?
- **完美主义**:每句话都想讲准
- **能量内耗**:大量精力花在"看起来怎样",真正讲的内容没多少

"对话"心态的好处:
- **关注外部**:听众在想什么?他需要什么信息?
- **真实**:讲不顺也不慌,**朋友之间允许卡壳**
- **能量集中**:精力花在内容上,不是形象上

**操作动作**:被点名站起来时,**找一个具体的人**——讲给他听,不是讲给"一群人"。

## Mindset 跟 [[板块核心心态]] 的关系 ⭐

这两条心法**本质就是核心心态在即兴场景的具体落地**:

| 即兴 Mindset | 在板块核心心态中对应 |
|---|---|
| **Dare to be dull** | **Congruence(内外一致)**——不演,真实就行 |
| **Conversation, not performance** | **更大的目标(I'm here to share)**——对话姿态,关注听众 |

> 没有读过 [[板块核心心态]] 直接看本页,Mindset 容易**只学到表面**。**心态根扎好,这两条心法才真正起作用**。

## 反例:错误的即兴心态

| 错误心态 | 后果 |
|---|---|
| "我必须讲精彩,不然丢人" | 想金句 → 卡壳 → 真丢人 |
| "我得讲到老板满意" | 自我审视 → 失去对听众的关注 → 讲偏 |
| "我要讲得跟 XX 一样厉害" | 模仿别人 → 不像自己 → Congruence 崩 |
| "我得显得专业" | 用复杂词汇 → 听众听不懂 → KISS 崩 |
| "我得讲完五分钟才不丢人" | 拖时间 → 废话堆砌 → 听众失耐心 |

**通用反例特征**:**关注的是"我"**——而不是"听众需要什么"。

## 心态调整的临场动作(被叫起来后 ~3 秒)

这 3 秒**只做心态重置**,不做选框架:

```
1. 深呼吸 1 次
2. 念一句 Mantra(来自 [[板块核心心态]]):
   "I'm here to share."
   "I have value to add."
3. 找一个听众的眼神
```

> **3 秒不算长,但够你重置心态**。

**重置完心态,才进入下一步**:用 [[结构-5框架|5 框架]] 那一节的「~5 秒判断问题类型 → 选一个微结构」,再开口。心态 3 秒 + 选框架 5 秒,都装在 [[0-即兴演讲总览|总览]] 的 **30 秒临场流程**里——三个数字是同一条流程的不同片段,不是互相打架。

```
[[心态-Mindset|本页]]:Mindset(心态稳)
        ↓
```

**两层缺一不可**:
- 只有 Mindset、没结构 → 心态稳但讲乱了
- 只有结构、没 Mindset → 套个框架但听起来生硬
