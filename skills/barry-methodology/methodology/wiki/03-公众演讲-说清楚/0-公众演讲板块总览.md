# 公众演讲 · 板块总览(说清楚)

> 公众演讲板块 = "说清楚"。以**听众为中心**,把已经"想清楚"的内容**重新编排 + 包装呈现**。**整个板块按演讲类型(半即兴 / 即兴)切分**,Mentality 作为开篇核心心态贯穿两类。

## 板块定位

```
英文沟通方法论
├── 00-学员心态铺垫(入学前认知准备)
└── 沟通的三层动作
    ├── 01-逻辑思维(想清楚)
    ├── 02-翻译能力(译清楚,英文场景特有的桥梁)
    └── 03-公众演讲(说清楚) ← 本板块
```

"说清楚"是"想清楚"的**下游板块**:

| 想清楚(逻辑思维) | 说清楚(公众演讲) |
|---|---|
| 以**问题**为中心 | 以**听众**为中心 |
| 追求**全**——不重不漏 | 追求**准**——听众要什么给什么 |
| 完整拆解问题 | 重新编排 + 包装呈现 |

进入公众演讲板块时,**内容已经有了**(由 [[0-逻辑思维板块总览]] 用 [[Phase2-金字塔结构]] 拆解完成)。本板块要解决的唯一问题是:

> **内容有了,怎么根据听众重新编排,让他们听进去、记得住、愿意行动。**

## 与"想清楚"的关系:加和减

同一套内容,进入演讲要做两件事:

| 操作 | 说明 |
|---|---|
| **加** | 加入 Hooks(**Ramp** = 开场钩子、**Dessert** = 情感收尾甜点 等)增强吸引力。逻辑思维不需要,演讲需要 |
| **减** | 对核心信息不重要的内容,略讲甚至删掉。逻辑思维要求完整,演讲要求精准 |

> **核心内容不变,编排方式变。**

## 整体框架(按演讲类型切分)⭐

```
公众演讲
├── 0. 基本定义
│   ├── 演讲四种类型(Manuscript / Memorization / Extemporaneous / Impromptu)
│   └── Performance Grid(内容 × 表现 四象限)
│
├── ⭐ 板块核心心态(Mentality)—— 整个板块的底色,贯穿两类演讲
│   ├── Congruence(内外一致 / In the moment)
│   ├── 与自己和解(紧张 = 兴奋,5 句 Mantra)
│   └── 更大的目标(I'm here to share)
│
├── 1. 半即兴演讲(Extemporaneous)—— 主体
│   ├── 1.1 内容 Content
│   │   ├── Communication Outline(Step 0-3)
│   │   ├── 演讲标准结构(Ramp → Roadmap → 3 PoDs → Q&A → Dessert)
│   │   ├── Techniques(Stories / Metaphors / Active language / Refrain)
│   │   ├── 高管汇报变体
│   │   └── Q&A 应对(Aggressive + Hostile 双套)
│   └── 1.2 传达 Delivery
│       ├── Physical/Vocal Image 理论锚
│       ├── 紧张应对(General principle + 5 句 Mantra + 具体症状)
│       ├── 声音(Voice — Melody:Volume / Pitch / Tempo + Pausing/Stressing)
│       └── 外在(Posture + Powersphere + Movement + Eye contact)
│
└── 2. 即兴演讲(Impromptu)—— 通用职场生存能力
    ├── 2.1 心态 Mindset
    │   ├── Dare to be dull(敢于平淡)
    │   └── Conversation, not performance(是对话不是表演)
    └── 2.2 结构 Structure(5 个口诀框架,调用 Mental Models 板块)
        ├── What — So What — Now What
        ├── PREP(Point — Reason — Example — Point)
        ├── Problem — Solution — Benefit
        ├── Comparison — Contrast — Conclusion
        └── STAR(Situation — Task — Action — Result)
```

> **核心术语速查**(详见 [[演讲标准结构]]):**Ramp** = 开场钩子段;**Roadmap** = 给听众的路线图;**PoD**(Point of Discovery)= 主体里的一个核心要点;**Dessert** = 情感收尾的"甜点"。这些是本板块的核心术语,首次出现先在此标注含义。

## 板块入口顺序

学员进入本板块的推荐阅读路径:

1. **[[演讲类型-Ext-Impromptu-Manuscript]]** —— 4 种演讲类型(认知框架)
2. **[[Performance-Grid]]** —— 内容 × 表现四象限(自我定位)
3. **⭐ [[板块核心心态]]** —— 演讲不是表演,是分享(底色)
4. **半即兴演讲** —— 主体训练
5. **即兴演讲** —— 通用生存能力

## 核心认知刷新

- **光有好内容不够,还要会讲**(来自 [[Performance-Grid]])
- **Extemporaneous 是目标演讲形态**(半即兴,不照稿、不全背)
- **Impromptu 是通用职场生存能力**——临场被叫上台讲话的应对
- **We make decisions based on our feelings**——Focus 不只包含 Know,还有 Feel 和 Do

## 在整个方法论中的位置

公众演讲板块是 [[MCES模型]] 链路 **落地的最后一段(S 节点输出)**:**想清楚 + 译清楚 + 说清楚 = 能真的把事说出去让人听进去**。

调用其他板块的能力:
- **[[0-逻辑思维板块总览|逻辑思维]]**:提供结构化内容(M)
- **[[0-翻译能力总览|翻译能力]]**:提供英文语言储备(C↔E)+ 软实力脱壳
- **[[Mental-Models-总览]]**:演讲沟通类框架(尤其 [[即兴演讲/结构-5框架|即兴 5 框架]])直接来自这里
