**整体逻辑说明**:
- **01 想清楚 / 03 说清楚** = 中英沟通通用能力(中文沟通也要想清楚 + 说清楚)
- **02 译清楚** = 英文场景**特有的中间桥梁**(中文沟通不需要)。它不跟 01/03 严格平行——而是处理"想清楚的内容如何用英文 articulate"
- **04 Mental Models** 真正跨切:服务 01(决策思考类调用)+ 03(即兴演讲类调用)。**不服务 02**——所以是"服务 01+03",不是"服务 3 大支撑"
- **Four Strands** 不在跨切层 —— 它只服务 02 / 硬实力 / 宏观维度,所以归入 02 子目录(2026-05-14 重构)
- **MCES / 语块 / 输入输出并行** 原属 00 底层认知(贯穿性地基),2026-05-14 按"真实主体归属"挪到 02 板块内部 —— 00 现在纯粹是"学员入学前心态铺垫"
