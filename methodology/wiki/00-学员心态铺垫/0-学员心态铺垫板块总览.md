> Barry 英文沟通方法论的**入口铺垫**——专注**学员入学前的认知准备**。**结构**:迷思破解(先讲什么是错的)→ 正确心态(建立正确预期)。**核心原理(MCES / 语块 / 输入输出并行)已挪到各支撑板块内部**(2026-05-14 重构后)。


## 本板块的边界 ⭐

**本板块只做学员心态铺垫**——破解错误认知 + 建立正确预期。

**不再包含的内容**(2026-05-14 重构):

| 原 00 / 3.x 节点 | 挪到哪 | 为什么 |
|---|---|---|
| **3.1 MCES 模型** | `02-翻译能力/MCES模型.md` | MCES 是 02 / 硬实力 / 微观维度的核心,主战场在 02 |

## 入口顺序建议

**新学员**(第一次接触 Barry 体系):
1. **迷思 1, 2, 3** —— 先打掉错误认知
2. **学习心态** —— 建立正确预期
3. → 进入 [[0-翻译能力总览|02-翻译能力]] 板块(学习 MCES、语块、Four Strands 等核心模型)
4. → 进入 [[0-逻辑思维板块总览|01-逻辑思维]] 和 [[0-公众演讲板块总览|03-公众演讲]] 板块

**老学员**(已在练习,遇到瓶颈):
- 想放弃 / 怀疑路径 → 回看 [[2.1-学习心态-没有捷径|学习心态]] + 隧道理论
- 怀疑学英语的意义 → 回看 [[1.2-迷思二-AI时代要不要学英语|迷思二:AI 时代]]

## 跟其他板块的关系

```
00-学员心态铺垫(本板块)
    │
    │ 提供心态前提 — 让学员愿意持续做
    ↓
三大支撑能力 + 跨切工具库 + 附属资源
    │
    │ 学员在实战中遇到的所有方法论
    │
    │ (心态出问题时,回到本板块再 review)
    ↓
反过来加深对学习路径的理解
```

**心态先于方法论**:技巧再好,学员不练就没用。本板块的核心价值是**让学员愿意持续投入**。
