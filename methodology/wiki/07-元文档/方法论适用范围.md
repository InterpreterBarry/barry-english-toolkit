# 方法论适用范围

> Barry 英文沟通方法论的**主要受众、核心策略、以及明确的未覆盖区域**。**不是所有人都适合这套方法论**——先对号入座,再决定要不要用。

## 主要受众:一句话定义

**已经有英语基础(学过 6+ 年)的中国成人学习者,尤其是职场人群。**

这群人的画像:

- 学过 10+ 年英语,有**大量被动词汇储备**
- 痛点是"**认识但用不出来**"——不是"没英文词",是"调不出来"
- 需要在**工作场景**下真实使用英语(会议、汇报、邮件、跨国协作)
- **不是零基础**,不是母语者,不是纯学术研究者

## WHAT:当前体系的核心策略

针对上述受众,方法论的 6 大核心策略:

### 1. 翻译思维 → 英语思维的渐进过渡

通过 [[MCES模型]] 的脚手架,**不排斥翻译思维**,而是承认它是中国学习者的起点,再通过训练逐渐跳过中文。

### 2. 语块优先,不是单词优先

详见 [[语块优先原则]]。已有被动词汇的学习者,真正的瓶颈是**搭配调用**——语块(multi-word units)是语言的真实使用单位。

### 3. 表达优先,不是发音优先

有 10 年基础的中国学习者,**基本音素已经熟悉**,问题在超音段(连读、弱读、节奏)和"认识但说不出来"。不再花大量时间专门练发音,而是**在表达积累和输出训练中附带提升发音**。

### 4. 输出驱动

视译 + 即兴口语 + 商务翻译——把"认识的词"逼到"用得出来"的唯一方法是**大量输出**。对应 [[Meaning-focused-Output]] 的 Swain 三功能。

### 5. 大量可理解输入

详见 [[Meaning-focused-Input]]。虽然这是 Barry 体系目前的最大缺口,但已明确纳入课外引导体系。

### 6. 限时输出训练

Retelling / Mini Presentation 加入逐轮提速(参考 [[4-3-2-技巧]])——没有时间压力就不算 fluency,详见 [[Fluency-Development]]。

## WHY:为什么要明确适用范围

**不是所有学习者画像都适合这套方法论**。越是强势的方法论越需要说清楚"**不适合谁**",否则:

- 纯粹零基础用这套 → 语块优先会缺砖头,表达优先会没发音地基
- 天然浸泡环境(海外移民)用这套 → 翻译思维反而是桎梏
- 学术英语用户用这套 → 职场场景导向不覆盖深度写作

**诚实划定边界,比声称"适合所有人"更有力量**。

## WHAT:当前体系未覆盖的板块

### 1. 纯粹零基础

从未接触过英语的成人 / 儿童。该人群需要**不同的起步路径**——见 [[零基础学习路径-总览]]。核心参考来源是 Gabriel Wyner《Fluent Forever》。

### 2. 学术英语(深度写作)

Barry 方法论聚焦**职场口语 + 邮件 + 汇报**,不覆盖学术论文写作、学术文献阅读。Calum 也明确:AI 时代,对 Barry 学员而言,写作技能正在从"产出"转向"编辑 AI 产出"。

### 3. 语言学考试(IELTS / TOEFL 应试)

本方法论不是应试方法论。虽然掌握了方法论后考试成绩会提升,但**不为考试优化**。

### 4. 文学 / 诗歌 / 修辞

美感 + 文体的英文表达不在覆盖范围。

### 5. 儿童早期语言习得

儿童语言习得机制与成人不同,不在方法论适用区。

## HOW:对号入座的决策树

```
你是不是中国成人?
  |
  是 → 你有 6+ 年英语基础?
  |     |
  |     是 → 主要用于职场场景?
  |     |    |
  |     |    是 → ✅ 适用 Barry 方法论
  |     |    否 → 部分适用(语块、MCES 通用)
  |     |
  |     否 → ❌ 用 [[零基础学习路径-总览]]
  |
  否 → ❌ 本方法论不为你优化
```

## 和其他方法论的关系

- **Paul Nation Four Strands**:学术框架,与 Barry 体系融合见 [[Four-Strands-与Barry方法论的融合]]
- **Gabriel Wyner Fluent Forever**:图片法,主要适用于零基础起步——不是对立,是互补
- **口译训练体系(AIIC / Monterey)**:Barry 的视译训练借鉴了口译方法,但受众和目标不同——Barry 学员不追求同传水平,只追求职场够用
