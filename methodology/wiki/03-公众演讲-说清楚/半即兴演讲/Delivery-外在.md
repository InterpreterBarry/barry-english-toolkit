# Delivery · 外在(Appearance)

> Delivery 第三维度(继 State 和 Voice 之后):**Posture + Powersphere + Movement + Eye contact**。核心理念:**外在是 Physical Image 的载体**——决定听众对你的第一波假设。

## WHAT:四个维度

| 维度 | 核心要点 |
|---|---|
| **Posture(站姿)** | 标准站立 + **Powersphere**(中心稳定区域) ⭐ |
| **Movement(舞台移动)** | 60% 中间 / 20% 左 / 20% 右,**移动要有理由** |
| **Eye contact(眼神交流)** | **One thought, one person**,每人约 3 秒 |
| **手势** | 自然配合语言节奏,不做无意义小动作 |

## WHY:为什么外在重要

**听众的判断过程**:

1. 你还没开口 → 先看你**站姿 / 表情**(3 秒内形成第一印象)
2. 你开口后 → 声音 + 外在**一起被接收**
3. 外在和声音不一致 → 听众会**信外在**(因为身体语言更难伪装)

**核心认知**:外在不是"锦上添花",是**[[板块核心心态|Congruence(内外一致)]]**的外部体现。

## HOW:Posture + Powersphere ⭐

### 标准站姿

- 双脚与肩同宽
- 重心**稳**(不晃 / 不重心偏移到一只脚)
- 肩膀放松,**不耸**
- 头略抬,目光平视前方

### Powersphere(力量场域)⭐ 来自飞书课件

**Powersphere 是站姿的"中心稳定区域"**——你的身体核心(胸腔到髋部)处于**放松但稳固**的状态。

**特征**:
- 呼吸顺畅(没被屏住)
- 核心稳定(不左右摇晃)
- 双臂自然下垂,**手势从这个区域发出**(而不是从身体外缘)

**反例**:
- 双手插口袋 → Powersphere 被遮挡 / 显得封闭
- 双手抱胸 → Powersphere 被遮挡 / 显得防御
- 来回晃动 → Powersphere 不稳

**为什么重要**:Powersphere 是**听众判断你"稳不稳"的视觉锚点**。它稳了,听众感觉**你这个人是稳的**——Physical Image 自然到位。

## HOW:Movement(舞台移动)

规则:**60% 中间 / 20% 左 / 20% 右**

**移动要有理由**:
- 讲到 PoD 1 → 站在左边
- 讲到 PoD 2 → 回到中间
- 讲到 PoD 3 → 站在右边
- 回到 Dessert → 回到中间

**绝对不要**:
- 来回踱步 —— 分散注意力
- 背对听众 —— 丢失连接
- 固定不动 —— 显得僵硬

## HOW:眼神交流

**核心法则:One thought, one person**

- 一个完整 thought(一句话或一个小段落)→ 看一个人
- 每人约 **3 秒**(不要一闪而过,也不要盯着一个人过久)
- 扫描整个房间,不要只看前排

**为什么是 3 秒**:
- < 1 秒 = 没建立连接,像在扫视
- > 5 秒 = 让对方不适,其他人觉得被冷落
- 3 秒 = 刚好让对方感到"他在看我"

**听众会看向你看的地方** —— 所以眼神是引导注意力的工具,不只是个人习惯。

## HOW:手势配合

- 手势自然随语言起伏,**在重点词附近**做手势
- 手不插口袋 / 不背手 / 不抱臂
- 不做**重复小动作**(扶眼镜、摸头发、抖腿)—— 这些是紧张信号

## HOW:着装

- 着装要**匹配场景**(路演穿正装,技术分享穿 smart casual)
- 不穿过于显眼 / 花哨的配饰 —— 让听众看内容,不是看你的装饰
- **舒适 > 时尚** —— 紧绷的衣服会影响呼吸和站姿

## 紧张应对中的外在维度

详见 [[Delivery-紧张应对]]。外在相关动作:
- 发抖 → 动一动,走两步,加手势(给紧张能量出口)
- 脸红出汗 → 握凉杯子(物理降温)
- 站不稳 → 重新找回 Powersphere(核心稳定)

## 站姿 / 移动 / 眼神的共同底层

这三个都指向同一件事:**让观众感受到"他在跟我们说话"**。

所有外在技巧最终都是为了**连接感**:
- 站姿稳 = 我在这里,没在飘
- 移动有理由 = 我在引导你的注意力
- 眼神接触 = 我看见你了

连接感的源头是 [[板块核心心态]]。
