# 切分

> [[中到英翻译-三步走]] 的 **Step 1**:找到一句中文里到底有几个核心意思,把它们一一拆出来。

## 核心观点 / WHAT

**一句中文 ≠ 一个 idea。** 中文习惯把多个意思塞进一句长话里。翻译前必须先把这些意思**切分出来**,一意一翻,再用连词串起来。

```
中文长句 → [切分] → 多个 idea
                      ↓ 每个 idea 单独进入 Step 2(脱壳 + 用 KISS 写一个小短句)
                      ↓ Step 3(用 [[逻辑连词显化|逻辑连词]] 串起来)
```

## WHY:为什么先切分

- 中文一句话**经常含 2-3 个独立意思**,不切分直接翻 → 英文写成超长复合句 → 翻译腔 + 听者抓不住重点
- 切分后**一意一短句**符合英文的"低密度、显逻辑"特征(详见 [[KISS原则]])
- 切分清楚后,**Step 3 加连词**就有了明确的接口——each idea is a node, 连词是 edge

## HOW:操作流程

1. 拿到中文,**先不翻**,问自己:"这句话里到底说了几件事?"
2. 把每件事用**一句中文**单独写下来(可以是大白话)
3. 每件事进入 Step 2(脱壳 + KISS 写一个英文小短句)
4. 用 because / but / so / and 等连词把这些小短句串成完整英文

## 示例

### 示例 1(2 个意思)

**中文**:精力过度分散,不聚焦,造成业务管理、团队管理的各种决策质量不高。

**切分**:
1. 精力过度分散,不聚焦
2. 造成业务管理和团队管理的决策质量不高

### 示例 2(2 个意思)

**中文**:我们也希望把这个成功经验复制到更多的腰尾部客户,并带动整个行业的发展。

**切分**:
1. 把经验复制到更多客户
2. 带动整个行业发展

### 示例 3(3 个意思)

**中文**:我们这边的客户反馈最近响应速度变慢,影响了整体的成单率,需要尽快解决。

**切分**:
1. 客户反馈响应速度变慢
2. 影响了成单率
3. 需要尽快解决
