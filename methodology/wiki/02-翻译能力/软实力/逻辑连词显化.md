# 逻辑连词显化

> 中文习惯把**逻辑关系隐含在文字里**,英文必须**把逻辑显性标出来**。翻译的第三步:补连词。

## 核心观点 / WHAT

**中文隐,英文显。** 同一段话,中文靠"上下文+语序"让人自己悟出逻辑,英文必须把 because / but / so 显性写出来。

```
中文  →  "任务很紧。人手不够。下周交不了。"
            (中国人自动读成:因为任务紧,而且人手不够,所以下周交不了)

英文  →  The task is urgent. We're short on staff. We can't deliver by next week.
            (英文母语者读完会问:所以呢?这三句有关系吗?)

        正确  →  The task is urgent, BUT we're short on staff, SO we can't deliver by next week.
```

## WHY:为什么中英差别这么大

### 语言类型学差异

- **中文**是"高语境"语言——大量含义靠上下文、语序、语气承载
- **英文**是"低语境"语言——每一层逻辑关系都要**显性标记**

**中国人读中文**,大脑自动补全逻辑;**英语母语者读英文**,没有连词就**默认是并列关系**——丢逻辑。

### 职场沟通的实际代价

- 中式英语里最典型的问题不是语法错,是**逻辑断层**
- 老板听不懂你的汇报,不是因为你单词错,是因为**你每一句都像独立陈述,他抓不住因果**

**修复成本**:加几个连词,表达一下子就清晰。

## HOW:四类必须显化的逻辑关系

| 逻辑关系 | 中文习惯 | 英文需要显化 |
|---|---|---|
| **因果** | 隐含不说 | because / so / therefore / as a result |
| **转折** | 隐含不说 | but / however / although / despite |
| **递进** | 隐含不说 | and / moreover / in addition / what's more |
| **条件** | 隐含不说 | if / unless / as long as / provided that |

### 其他高频逻辑

| 关系 | 连词 |
|---|---|
| 举例 | for example / such as / like |
| 让步 | although / even though / while |
| 目的 | so that / in order to |
| 总结 | in short / to sum up |

### 操作流程

1. **Filter 后**,盯着核心信息,问自己:**每两句之间的关系是什么?**
2. 用 4 类基础关系(因果/转折/递进/条件)套一遍
3. 补上合适的连词
4. 句子自然会从"chunk"变"sentence"

## 示例:会议纪要的逻辑显化

### 原始中文(Filter 后)

> 本月销售目标 1000 万。实际完成 600 万。市场整体需求下降。团队士气受挫。下月要调整策略。

### 如果不显化逻辑

> *Monthly sales target: 10 million. Actual: 6 million. Market demand declined. Team morale is low. We need to adjust strategy next month.*

这是**列事实**,不是**汇报**——老板听完会反问"所以呢?"

### 显化逻辑后

> *We set a sales target of 10 million this month, **but** we only hit 6 million—**mainly because** market demand declined, **which** hit team morale. **So** we need to rework our strategy next month.*

**差别**:
- but(转折:目标 vs 实际)
- mainly because(因果:业绩差 ← 需求下降)
- which(递进因果:需求下降 → 士气)
- so(因果:所以要调整)

**同样的信息**,显化后变成一段**有逻辑有叙事**的汇报。

## 常见误区

| 误区 | 问题 |
|---|---|
| 连词越多越好 | 不,连词过度会让英文显得啰嗦。**关键逻辑点**显化,不是**每句**都加 |
| 只用 and | and 是最弱的连词,大部分场合该用 but / so / because,而不是 and |
| "中文没说,我不敢加" | 中文没说**不是"不存在"**,是"隐含"。翻译时你必须**把它挖出来显化** |
