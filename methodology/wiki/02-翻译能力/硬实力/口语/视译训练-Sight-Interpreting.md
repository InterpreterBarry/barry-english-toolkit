# 视译训练(Sight Interpreting)

> 看着中文材料,口头翻译成英文。**从翻译思维过渡到英语思维的核心训练方法**,也是输入与输出的综合训练。

## 核心定位

- 视译是 [[MCES模型]] 中把 M→C→E 链路打通的主力训练
- 目标不是产出漂亮译文,而是**通过反复训练,逐步让大脑跳过中文中介**
- 是 Barry 方法论里解决"主动词汇太少"这个核心口语问题的**首选方案**

## WHAT:视译是什么

看一段中文,**不写、直接口头说英文**。材料以句子为单位,每天半小时,练几句话或一个小段落。

**漏斗模型**:
```
被动词汇(看得懂/听得懂,但用不出来)
        ↓ 视译训练
主动词汇(能随时调动使用)
```

## WHY:为什么视译是核心方法

Barry 方法论对"口语不好"的诊断只有一句话——**主动词汇太少**。

| 词汇类型 | 定义 |
|---|---|
| 被动词汇 | 看到/听到能认识,但用不出来 |
| 主动词汇 | 能随时调动使用 |

中国学员普遍是"学了 10+ 年英语,认识但用不出来"。单纯再背单词不解决问题——问题不在储备,在**调动**。视译用中文材料作为**调动触发器**,强迫大脑从被动储备里把词汇"拽出来"使用。

## HOW:操作步骤

1. **拿到中文材料**(以句子为单位)
2. **找出中文固定搭配/语块**([[语块概念-Meaning-Unit]])
3. **对照英文翻译,筛选中英文语块**
4. **根据中文语序组织英文句子**——早期允许中式语序,随着训练推进逐步摆脱

### 常见问题与纠偏

| 问题 | 表现 | 正确做法 |
|---|---|---|
| 字对字翻译 | 逐字对应,生硬直译 | 先做 [[脱壳-Deverbalization|脱壳]],理解整体 Message 后用英文重新表达 |
| 只对应语块,忽略意思 | 机械匹配中英语块,没真正理解 | 先理解 M,再连接 E,语块只是桥梁 |

**核心提醒**:视译的目标是打通"意思→英文"的通路,语块只是帮助你过渡的工具,不要停留在"中文语块→英文语块"的机械对应上。

## 示例

**中文材料**:"供应商和生产线的数据相关性比较薄弱"

- **[[脱壳-Deverbalization|脱壳]]后**:两边的数据对不上
- **英文重新表达**(因人而异):
  - 基础:*The data from A and B don't match.*
  - 较高:*The two teams are not using the same data metrics.*

## 练习配置

- **频率**:每天半小时
- **量级**:几句话或一个小段落
- **材料**:职场英语相关内容(保证场景贴合)

## 视译 ≠ 背诵译文 ⭐

视译训练的**精髓**是"自己拼句子的过程",**不是背英文译文**。详细心法见 [[中英语块加视译]]。

## 跟"中英语块 + 视译"的关系

视译这个**动作**,跟"中英语块"这个**弹药库**绑在一起,才构成完整的训练方法。本页讲视译动作的机制、操作、纠偏;完整训练方法的统一理解见 [[中英语块加视译]]。
