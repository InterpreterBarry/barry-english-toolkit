# 4U 原则(标题设计)

> 大型登台演讲的标题设计四原则:**U**sefulness / **U**niqueness / **U**rgency / **U**se slogan。
> 在"想清楚"板块中,是 [[Phase2-金字塔结构|Phase 2]] HOW 章节的**落地工具之一**。

## 核心

**标题不是装饰,是金字塔的塔尖露出水面的那一小块。** 听众还没进入内容,先通过标题判断「这个值不值得我听下去」——4U 就是判断标准的四个维度。

## WHAT:4U 拆开讲

| 原则 | 含义 | 一句话测试 |
|------|------|----------|
| **Usefulness**(有用) | 受众通过标题知道**能获得什么** | 听完这场,我能拿走什么? |
| **Uniqueness**(独特) | 告诉受众**有何特殊之处** | 别人讲这话题讲过了,你凭什么? |
| **Urgency**(紧迫) | 塑造**紧迫感和危机感** | 为什么是今天听而不是明年? |
| **Use slogan**(口号化) | 口号式,唤起**责任心和使命感** | 能不能让人听完想立刻转发/行动? |

## 适用范围

> **4U 不是万能标题公式,只用在「大型登台演讲」场景。**

- ✅ 线下大课、行业大会、百人以上演讲、开源方法论发布会
- ❌ 日常邮件、工作汇报、一对一沟通、小组例会

**原因**:4U 要求在标题里挤进四个维度,信息密度高、戏剧感强——**和日常沟通的 KISS 原则冲突**。大场合要「抓眼球」,小场合要「说清楚」,不是一回事。

## WHY:为什么是这 4 个维度而不是别的

- **Usefulness 解决「听了有啥用」** → 受众的**理性决策依据**
- **Uniqueness 解决「凭什么听你」** → 和其他讲同话题的人**差异化**
- **Urgency 解决「为什么是现在」** → **激活行动意愿**
- **Use slogan 解决「听完怎么记」** → **传播性**,听众会帮你传出去

四个合起来,**理性 + 差异 + 行动 + 传播** 全齐了——这正是**大场演讲 KPI** 的四个指标。

## HOW:从普通标题升到 4U 标题

### 示例:关于职场英语学习的演讲

**普通标题**:
> 职场英语学习的几个方法

对照 4U:
- Usefulness:弱(「几个方法」不具体)
- Uniqueness:无(谁都在讲)
- Urgency:无(为什么今天听?)
- Use slogan:无(记不住)

**4U 升级版**:
> **90 天说一口能谈判的职场英语:同传不会告诉你的 3 件事**

- Usefulness:✅ 能「谈判」,不是泛泛而谈
- Uniqueness:✅ 「同传不会告诉你」(独家视角)
- Urgency:✅ 「90 天」(明确时间窗)
- Use slogan:✅ 「3 件事」(可复述、易传播)

## 4U 和 SCQA 的配合

**标题 = 塔尖的第一次亮相;SCQA 序言 = 塔尖的第二次亮相。**

```
标题(4U)                  → 让他决定「进不进门」
    ↓
SCQA 序言(S-C-Q-A)         → 让他决定「留不留下来」
    ↓
主体(金字塔结构)           → 让他「被说服」
```

两个都到位,开场 2 分钟里就把听众「抓住」了。

## 检查:我的标题真的 4U 吗?

拿到标题后,问自己四个问题:

1. 受众看到标题,**立刻知道自己能得什么**吗?(Usefulness)
2. 他**为什么不去听别人讲这个**?(Uniqueness)
3. 为什么**现在**就要听?(Urgency)
4. 他听完后**能一句话告诉别人**这讲了啥吗?(Use slogan)

四个全「是」 → 过关。任何一个「否」 → 回去改。
